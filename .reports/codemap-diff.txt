ARCHITECTURE CHANGES
====================

Dispersant Selection (Pipeline J) — Dual-HSP Model:
  Unique among all pipelines: uses TWO separate HSP evaluations per dispersant.
  1. Anchor HSP → particle surface affinity (RED_anchor)
  2. Solvation HSP → solvent compatibility (RED_solvation)
  3. Composite RED = √(RED_anchor × RED_solvation) — geometric mean
  This requires the Dispersant type to carry 6 HSP values (3 anchor + 3 solvation)
  instead of the standard 3 HSP values used by other entities.

Pipeline Letter Reassignment:
  Previous J-Q (Advanced) → Now K-R
  This maintains alphabetical continuity with the new J (Dispersant Selection)
  inserted into Core Classifiers group.
