# 第9章 医薬品とドラッグデリバリー — HSPによる合理的製剤設計

**Chapter 9: Pharmaceutical Drug Delivery — Rational Formulation Design via HSP**

> **概要:** 本章では、Hansen溶解度パラメータ（HSP）を医薬品製剤およびドラッグデリバリーシステムの設計に応用する手法を体系的に解説する。REDに基づく薬物溶解性分類から出発し、経皮吸収促進剤の選定における三成分HSPシステム（薬物-皮膚-促進剤）、リポソーム膜透過性予測、ナノ粒子薬物担持の最適化、共結晶スクリーニング、多形・溶媒和物リスク評価、固体分散体の担体賦形剤選定、吸入製剤におけるプロペラント適合性評価まで、医薬品開発の各段階におけるHSPの統合的活用法を論じる。HSPフレームワークは、これら多岐にわたる製剤課題に対して統一的な熱力学的基盤を提供し、網羅的実験スクリーニングの必要性を大幅に低減する。

---

## 学習目標

本章を修了した読者は、以下の能力を獲得することを目標とする。

1. RED分類を適用して医薬用溶媒中の薬物溶解性を予測し、API溶解のための候補溶媒を合理的にランク付けできる
2. 三成分HSP解析（薬物-皮膚-促進剤）を用いて化学的浸透促進剤を選定し、経皮ドラッグデリバリー製剤を設計できる
3. 薬物と担体高分子のHSPマッチングからナノ粒子薬物担持効率を予測し、カプセル化を最適化できる
4. HSP距離基準を用いてAPI-コフォーマー対の共結晶形成可能性をスクリーニングし、有望な候補を同定できる
5. REDトランジションゾーン（0.5--1.5）の溶媒を同定することにより、結晶化工程における多形・溶媒和物リスクを評価できる

## 9.1 医薬品製剤とHSP — 歴史的展開と理論的基盤

### 9.1.1 医薬品分野におけるHSPの導入

溶解度パラメータの医薬品システムへの適用は、Hildebrandの一成分溶解度パラメータに遡る。Hancockら（1997）は、賦形剤マトリクス中の薬物溶解性を予測するための溶解度パラメータの体系的利用を提案し、医薬品製剤設計への定量的アプローチの端緒を開いた [Hancock et al., 1997]。しかし、一成分パラメータでは極性相互作用や水素結合の寄与を分離できず、特に多官能基を持つ薬物分子の溶解挙動を十分に記述できないという限界があった。

Hansenの三成分拡張は、この問題を根本的に解決した。Archer（1992）が医薬品溶媒選択にHSPを適用した先駆的研究を発表し、その後Gharagheizi（2008）およびMohammadら（2011）によって体系化された [Mohammad et al., 2011]。AbbottとHansen（2008）は、HSP球法と RED分類を医薬品溶媒スクリーニングに形式化し、薬物の溶解性予測における統一的枠組みを確立した [Abbott & Hansen, 2008]。

経皮吸収促進剤選定へのHSP適用は、Fangら（2003）およびMinghettiら（2007）によって探索され、三成分薬物-皮膚-促進剤アプローチが実証された。共結晶スクリーニングへのHSP距離の利用は、Greenhalghら（1999）が固体分散体の相溶性予測においてHSP距離とAPI-コフォーマー間の共結晶形成傾向の相関を示し [Greenhalgh et al., 1999]、Mohammadら（2011）がこれを発展させた。多形・溶媒和物リスクゾーンの概念（RED 0.5--1.5）は、部分的溶解性が結晶形多様性を促進する条件を生むという観察に基づいており、Threlfall（2000）およびBernstein（2002）によって議論された [Bernstein, 2002]。ナノ粒子薬物担持のHSPマッチングによる最適化は、LetchfordとBurt（2007）の研究から発展し、Duongら（2015）によってさらに精緻化された [Letchford & Burt, 2007]。

現在、計算論的HSP予測手法の進歩により、薬物製剤候補のバーチャルスクリーニングが可能となり、実験コストの大幅な削減が実現されている。

### 9.1.2 HSP距離とRED — 薬物溶解性の基本方程式

薬物-溶媒対のHSP距離 $Ra$ は、三次元HSP空間における二点間距離として定義される。

$$
Ra = \sqrt{4(\delta_{D,drug} - \delta_{D,solvent})^2 + (\delta_{P,drug} - \delta_{P,solvent})^2 + (\delta_{H,drug} - \delta_{H,solvent})^2}
$$

> Unicode: Ra = sqrt[4(δ_D,drug − δ_D,solvent)² + (δ_P,drug − δ_P,solvent)² + (δ_H,drug − δ_H,solvent)²]

ここで、$\delta_D$、$\delta_P$、$\delta_H$ はそれぞれ分散力、極性、水素結合の溶解度パラメータ成分である。分散力項の係数4は、HSP球の球形近似を最適化するための経験的重みであり（第1章参照）、薬物分子に対しても同様に適用される。$Ra$ が小さいほど薬物-溶媒間の分子間相互作用が類似しており、高い溶解性が期待される。

薬物溶解性の分類は、相対エネルギー差（RED）により定量化される。

$$
RED = \frac{Ra}{R_0}
$$

> Unicode: RED = Ra / R₀

ここで $R_0$ は薬物のHSP球の半径であり、実験的溶解度データから決定される。REDによる分類基準は以下のとおりである。

| RED範囲 | 分類 | 意味 |
|---------|------|------|
| RED < 1.0 | 良溶媒 | 薬物が溶解する可能性が高い |
| RED = 1.0 | 境界 | HSP球の表面上 |
| RED > 1.0 | 貧溶媒 | 薬物が溶解しない可能性が高い |

**HSP球の構築.** 薬物のHSP球は、複数の溶媒に対する実験的溶解度データから構築される。各溶媒を「良溶媒」または「貧溶媒」に二値分類し、三次元HSP空間において良溶媒群を包含する最小球を求める最適化により、球の中心座標（$\delta_D$, $\delta_P$, $\delta_H$）と半径 $R_0$ が決定される。この手法は第1章で導入したHSPiP法の薬物分子への直接的応用である。

### 9.1.3 生物薬剤学分類システム（BCS）とHSP

生物薬剤学分類システム（Biopharmaceutics Classification System, BCS）は、薬物を水溶性と膜透過性の2軸で4クラスに分類する。HSPに基づく溶解性予測は、特にBCS Class II（低溶解性・高透過性）の薬物に対する製剤戦略の合理的設計に有用である。

| BCSクラス | 溶解性 | 透過性 | HSPとの関連 |
|-----------|--------|--------|-------------|
| Class I | 高 | 高 | 水のHSPに近い。製剤上の課題は少ない |
| Class II | 低 | 高 | 水のHSPから遠い。溶解性改善が主要課題 |
| Class III | 高 | 低 | 皮膚脂質ドメインのHSPから遠い |
| Class IV | 低 | 低 | 水・脂質双方のHSPから遠い |

BCS Class IIの薬物は、分子の疎水性が高く水のHSP（$\delta_D = 15.5$, $\delta_P = 16.0$, $\delta_H = 42.3$ MPa$^{1/2}$）から大きな $Ra$ を持つ。これらの薬物に対して、HSP解析は共結晶化、固体分散体形成、ナノ粒子封入など複数の溶解性改善戦略の合理的選択を可能にする。

## 9.2 医薬品溶媒スクリーニング — HSP球法の応用

### 9.2.1 薬物のHSP値決定

薬物のHSP値は、複数の手法により推定または実験的に決定される。

1. **実験的HSP球法**: 30--50種の溶媒に対する溶解度試験を実施し、良溶媒/貧溶媒の二値分類からHSP球を構築する
2. **グループ寄与法**: 分子構造中の官能基からHSP値を推算する（第4章参照）
3. **逆ガスクロマトグラフィー（IGC）**: 固体薬物表面との溶媒蒸気の相互作用から表面HSPを決定する
4. **分子シミュレーション**: 分子動力学計算から凝集エネルギー密度を算出してHSPを推定する

### 9.2.3 過飽和と核生成傾向

溶媒HSPがAPIのHSP球境界近傍（RED ≈ 0.8--1.2）に位置する場合、薬物は中程度の溶解度を示し、温度変化や貧溶媒添加によって過飽和状態に容易に到達する。この領域の溶媒は、結晶化プロセスにおける核生成と結晶成長の制御に重要な役割を果たす。

RED < 0.5の良溶媒からの結晶化では、溶解度が高いため大量の貧溶媒を添加する必要があり、核生成速度の制御が困難になりやすい。一方、RED ≈ 0.7--0.9の溶媒は、適度な溶解度と制御可能な過飽和度を両立するため、結晶化溶媒として実用的に好ましい場合が多い。

## 9.3 経皮ドラッグデリバリー — 三成分HSPシステム

### 9.3.1 経皮吸収の課題とHSP

経皮ドラッグデリバリーシステム（TDDS）は、皮膚を通じた薬物の全身投与を実現する投与経路である。最大の障壁は角層（stratum corneum）の脂質バリアであり、多くの薬物はこのバリアを十分な速度で透過できない。

化学的浸透促進剤（Chemical Permeation Enhancer, CPE）は、角層脂質の構造を一時的に乱すことにより薬物透過性を向上させる。CPEの選定において、HSPは薬物・皮膚・促進剤の三者間の分子間相互作用を統一的に記述する強力なツールとなる。

角層脂質ドメインのHSP値は、複数の研究から以下の範囲にあることが報告されている。

- $\delta_D$ = 16.0 MPa$^{1/2}$
- $\delta_P$ = 3.5 MPa$^{1/2}$
- $\delta_H$ = 6.0 MPa$^{1/2}$

これらの値は、角層がセラミド・コレステロール・脂肪酸の混合脂質からなる比較的疎水的なバリアであることを反映している。

### 9.3.2 三成分HSP距離と促進メカニズム

経皮吸収における促進因子（Enhancement Factor, EF）は、薬物・皮膚・促進剤の三成分間のHSP距離に関連づけられる。以下の半経験的関係が提案されている。

$$
\ln EF \propto -(Ra_{enhancer-skin})^2 + (Ra_{drug-skin})^2 - (Ra_{drug-enhancer})^2
$$

> Unicode: ln EF ~ −(Ra_enhancer-skin)² + (Ra_drug-skin)² − (Ra_drug-enhancer)²

この関係は、以下の物理的解釈に基づく。

- **第1項** $-(Ra_{enhancer-skin})^2$: 促進剤と皮膚脂質のHSP距離が小さいほど（HSPが近いほど）、促進剤は角層脂質ドメインに浸透しやすく、バリア構造を効果的に乱す
- **第2項** $(Ra_{drug-skin})^2$: 薬物と皮膚のHSP距離が大きいほど、促進剤なしでの透過が困難であり、促進効果の余地が大きい
- **第3項** $-(Ra_{drug-enhancer})^2$: 薬物と促進剤のHSP距離が小さいほど、促進剤が薬物の溶媒和を補助し、角層内での薬物の熱力学的活量を維持する

> **注意:** この式は半経験的モデルであり、定量的予測精度には限界がある。促進剤の角層内での動態、薬物の分子量・解離定数、皮膚の個体差など、HSPでは捉えきれない因子が実際の透過速度に影響する。

**実用的な促進剤選定の指針.** HSP解析に基づく促進剤選定では、以下の戦略が有効である。

1. **バリア親和型促進剤**: 角層脂質とのHSP距離が小さい促進剤（オレイン酸、Azone、リモネン）を第一候補とする
2. **薬物溶媒和型促進剤**: 薬物とのHSP距離が小さい促進剤を組み合わせて、角層内での薬物溶解度を向上させる
3. **二元系促進剤**: バリア親和型と薬物溶媒和型の促進剤を組み合わせることで、相乗的な促進効果を実現する

## 9.4 ナノ粒子薬物担持 — 高分子-薬物HSPマッチング

### 9.4.1 ナノ粒子DDSの原理

高分子ナノ粒子は、難水溶性薬物の溶解性改善、標的指向性、持続放出を実現するドラッグデリバリーキャリアとして注目されている。薬物担持量（Drug Loading Capacity, DLC）と封入効率（Encapsulation Efficiency, EE）は製剤設計の中核的パラメータであり、これらは薬物と担体高分子の熱力学的相溶性に直接依存する。

### 9.4.2 薬物-高分子間のFlory-Huggins相互作用パラメータ

薬物-高分子対の相溶性は、Flory-Huggins相互作用パラメータ $\chi$ により定量化される。HSPからの推算式は以下のとおりである。

$$
\chi_{drug-polymer} = \frac{V_{drug}}{RT} \left[ (\delta_{D,drug} - \delta_{D,poly})^2 + 0.25(\delta_{P,drug} - \delta_{P,poly})^2 + 0.25(\delta_{H,drug} - \delta_{H,poly})^2 \right]
$$

> Unicode: χ_drug-polymer = (V_drug / RT) × [(δ_D,drug − δ_D,poly)² + 0.25(δ_P,drug − δ_P,poly)² + 0.25(δ_H,drug − δ_H,poly)²]

ここで $V_{drug}$ は薬物のモル体積、$R$ は気体定数、$T$ は絶対温度である。極性項・水素結合項の係数0.25は、第8章で導入したLindvigらの経験的重みと同一であり、エネルギー論に基づく熱力学量の推算に適用される。

$\chi$ 値が小さいほど薬物-高分子間の親和性が高く、薬物担持量の増大が期待される。一般に $\chi < 0.5$ で良好な相溶性、$\chi > 2.0$ で非相溶と判定される。

### 9.4.3 薬物担持量のHSP距離依存性

薬物担持量はHSP距離と以下の指数関数的関係を示す。

$$
DLC \propto \exp\left(-\frac{V_{drug} \cdot (Ra_{drug-polymer})^2}{RT}\right)
$$

> Unicode: DLC ~ exp(−V_drug × Ra_drug-polymer² / RT)

この関係は、薬物-高分子間のHSP距離の二乗が大きくなるほど、担持量が指数関数的に減少することを示す。分子論的には、$Ra^2$ は薬物と高分子マトリクスの混合エンタルピーに比例し、$Ra$ の増大は薬物のナノ粒子コアからの相分離（結晶化）を促進する。

> **注意:** この関係は理想的な非晶質薬物-高分子混合系を前提としており、結晶性薬物の融解エンタルピーは考慮していない。実際の系では結晶性の影響を補正する必要がある。

### 9.4.4 キャリア選定の実践的指針

HSPマッチングに基づくキャリア選定では、以下の手順が推奨される。

1. 対象薬物のHSP値を決定する（表9.1参照、またはグループ寄与法で推算）
2. 候補高分子群とのHSP距離 $Ra$ および $\chi$ パラメータを計算する
3. $Ra$ の小さい順（または $\chi$ の小さい順）にキャリアをランク付けする
4. 上位候補について実験的検証を行い、担持量・封入効率・放出プロファイルを最適化する

例えば、イブプロフェン（$\delta_D = 17.4$, $\delta_P = 4.0$, $\delta_H = 9.6$）に対しては、PCL（$\delta_D = 17.0$, $\delta_P = 4.7$, $\delta_H = 8.3$）が最もHSP距離が小さく、高い担持量が予測される。一方、キトサンは水素結合パラメータが大きく異なるため、イブプロフェンとの相溶性は低い。

## 9.5 共結晶スクリーニング — API-コフォーマーHSP距離

### 9.5.1 共結晶の意義と課題

共結晶（cocrystal）は、APIとコフォーマー分子が化学量論的比率で結晶格子を形成する多成分結晶であり、APIの溶解性・バイオアベイラビリティ・安定性の改善手段として近年注目されている。特にBCS Class IIの難水溶性薬物において、共結晶化は有効な製剤戦略である。

しかし、適切なコフォーマーの選定は試行錯誤的であり、100種以上のコフォーマーを逐一スクリーニングする必要がある場合も少なくない。HSP距離に基づくスクリーニングは、有望なAPI-コフォーマー対を効率的に予備選択する手段を提供する。

### 9.5.2 共結晶形成とHSP距離基準

API-コフォーマー間のHSP距離は以下のように定義される。

$$
\Delta HSP_{API-coformer} = \sqrt{(\delta_{D,API} - \delta_{D,cof})^2 + (\delta_{P,API} - \delta_{P,cof})^2 + (\delta_{H,API} - \delta_{H,cof})^2}
$$

> Unicode: ΔHSP_API-coformer = sqrt[(δ_D,API − δ_D,cof)² + (δ_P,API − δ_P,cof)² + (δ_H,API − δ_H,cof)²]

ここで注意すべきは、この距離関数にはHansen距離 $Ra$ における分散力項の係数4が含まれていないことである。共結晶形成は溶解性とは異なる現象であり、分散力・極性・水素結合の三成分が等重みで寄与するモデルが経験的により良い予測を与えることが示されている [Mohammad et al., 2011]。

**経験的基準.** Greenhalghら（1999）およびMohammadら（2011）の研究に基づき、以下の基準が提案されている。

- $\Delta HSP < 5$ MPa$^{1/2}$: 共結晶形成の可能性が高い（相互作用が類似しすぎて固溶体を形成する場合もある）
- $5 \leq \Delta HSP \leq 10$ MPa$^{1/2}$: 共結晶形成に最適な範囲（相補的だが同一ではない分子間相互作用）
- $\Delta HSP > 10$ MPa$^{1/2}$: 共結晶形成の可能性が低い（分子間相互作用が乖離しすぎている）

この「中間距離」基準の物理的背景は興味深い。共結晶形成には、API-コフォーマー間の特異的分子間相互作用（水素結合、$\pi$-$\pi$スタッキング等）が必要であるが、両者のHSPが完全に一致すると固溶体（solid solution）の方が熱力学的に安定となり、新たな結晶格子を形成する駆動力が失われる。逆に $\Delta HSP$ が過大であると、分子間相互作用が弱すぎて結晶格子を安定化できない。

## 9.6 多形・溶媒和物リスク評価 — REDトランジションゾーン

### 9.6.1 多形と溶媒和物の問題

医薬品結晶化プロセスにおいて、意図しない結晶多形の出現や溶媒和物の形成は、溶解性・バイオアベイラビリティ・安定性の変動を引き起こし、製品品質に深刻な影響を与える。リトナビルの多形転移事件（1998年）は、このリスクの重大性を医薬品業界に知らしめた代表的事例である。

### 9.6.2 REDリスクゾーンの定義

溶媒がAPIに対して持つRED値と多形・溶媒和物リスクの間に、以下の経験的関係が提案されている。

$$
0.5 \leq RED_{solvent-API} \leq 1.5
$$

> Unicode: 0.5 ≤ RED_solvent-API ≤ 1.5

この「トランジションゾーン」に位置する溶媒は、多形転移や溶媒和物形成のリスクが高い。物理的解釈は以下のとおりである。

- **RED < 0.5（溶解ゾーン）**: 薬物が十分に溶解するため、結晶化が生じにくい。溶解した状態では多形の問題は生じない
- **0.5 ≤ RED ≤ 1.5（リスクゾーン）**: 薬物が部分的に溶解する領域であり、溶解-再結晶の平衡が存在する。この条件下では、溶媒分子が結晶格子に取り込まれて溶媒和物を形成したり、準安定多形が動力学的に有利に核生成されたりする可能性がある
- **RED > 1.5（安全ゾーン）**: 薬物がほとんど溶解しないため、溶媒との相互作用が弱く、溶媒和物形成のリスクが低い。ただし懸濁状態での多形転移の可能性は残る

### 9.6.4 結晶化プロセスにおけるリスク最小化戦略

HSP解析に基づく多形リスク最小化のための結晶化戦略は以下のとおりである。

1. **溶媒選択**: REDリスクゾーン（0.5--1.5）を避け、RED < 0.5の良溶媒からの貧溶媒添加結晶化（anti-solvent crystallization）を検討する
2. **貧溶媒選択**: RED > 1.5の安全ゾーン溶媒を貧溶媒として用いる
3. **溶媒和物リスクの追加評価**: リスクゾーン溶媒を使用する場合は、溶媒の分子サイズ・形状が結晶格子に取り込まれやすいかを併せて評価する
4. **シード結晶の使用**: 目的多形のシード結晶を添加し、核生成を制御する

## 9.7 非晶質固体分散体（ASD）の安定性予測

### 9.7.1 ASDの原理とHSP

非晶質固体分散体（Amorphous Solid Dispersion, ASD）は、結晶性薬物を高分子マトリクス中に非晶質状態で分散させることにより、見かけの溶解度と溶出速度を劇的に改善する製剤技術である。しかし、非晶質状態は熱力学的に準安定であり、長期保存中に薬物が再結晶化して溶解性の利点が失われるリスクがある。

薬物-高分子間のHSP距離は、ASDの物理的安定性（結晶化開始までの時間）と強い相関を示す。HSP距離が小さいほど薬物-高分子間の分子間相互作用が強く、非晶質状態の安定化が期待される。

Greenhalghら（1999）は、薬物-高分子間のHSP距離（$\Delta \delta_t = |\delta_{t,drug} - \delta_{t,polymer}|$、ここで $\delta_t$ は全溶解度パラメータ）に基づく以下の基準を提案した [Greenhalgh et al., 1999]。

- $\Delta \delta_t < 7.0$ MPa$^{1/2}$: 相溶性が高く、安定なASDの形成が期待される
- $\Delta \delta_t > 10.0$ MPa$^{1/2}$: 非相溶であり、ASD形成は困難

Forsterら（2001）は、ホットメルトエクストルージョンによるASD形成において、溶解度パラメータ計算と熱分析を組み合わせた賦形剤選定手法の有効性を実証した [Forster et al., 2001]。

[図9.6: 非晶質固体分散体の安定性マップ — 薬物-高分子HSP距離と物理的安定性（結晶化開始までの時間）の関係をプロットし、HSPマッチングされた系の安定性の優位性を示す]

### 9.7.2 ASD用高分子キャリアの選定

ASD用高分子キャリアの選定においては、HSP距離に加えて以下の因子を考慮する必要がある。

1. **ガラス転移温度 ($T_g$)**: 高い $T_g$ を持つ高分子は、非晶質薬物の分子運動を制限し再結晶化を抑制する
2. **特異的分子間相互作用**: 水素結合供与・受容能が薬物と相補的な高分子が好ましい（HSPの $\delta_H$ 成分で評価可能）
3. **吸湿性**: 水分の取り込みは $T_g$ を低下させ、再結晶化を促進する
4. **溶出時の過飽和維持能**: 「スプリング・パラシュート」機構における高分子の結晶化抑制効果

## 9.8 リポソーム膜透過性とHSP

### 9.8.1 リポソーム製剤とHSPの関連

リポソームはリン脂質二重膜からなるナノスケル小胞であり、親水性薬物を水相コアに、疎水性薬物を脂質二重膜内に封入できる汎用的なDDSキャリアである。薬物のリポソーム膜透過性は、薬物とリン脂質のHSP親和性に依存する。

リン脂質（例: DPPC）のHSP値は、脂質アシル鎖部分と極性頭部の二領域で異なるが、二重膜全体としてはおおよそ以下の範囲にある。

- アシル鎖領域: $\delta_D \approx 16.5$, $\delta_P \approx 2.5$, $\delta_H \approx 3.0$ MPa$^{1/2}$
- 極性頭部領域: $\delta_D \approx 17.0$, $\delta_P \approx 10.0$, $\delta_H \approx 15.0$ MPa$^{1/2}$

薬物のリポソーム二重膜への分配は、主としてアシル鎖領域との HSP 親和性に支配される。$Ra_{drug-acyl}$ が小さい薬物は二重膜内に高濃度で蓄積するが、あまりに親和性が高いと放出速度が遅くなりすぎる。最適な封入と制御放出を両立するHSP距離の範囲が存在する。

### 9.9.1 定量噴霧式吸入器（MDI）とHSP

|------------|-----------|-----------|-----------|------|
| HFA-134a | 12.2 | 7.5 | 5.0 | 現行主流プロペラント |
| HFA-227ea | 12.0 | 6.0 | 4.2 | 代替プロペラント |
| CFC-12 (旧) | 12.3 | 2.0 | 0.0 | 規制により廃止 |

単位: MPa$^{1/2}$

HFAプロペラントはCFCと比較して $\delta_P$ と $\delta_H$ が著しく大きく、これはHFA分子の極性C-H結合および弱い水素結合能に起因する。このHSPの差異が、CFC製剤からHFA製剤への移行時に薬物の溶解性・懸濁安定性が変化した主因である。

[図9.8: MDIプロペラント-薬物適合性チャート — 代表的吸入薬物とプロペラント（HFA-134a, HFA-227ea）間のHSP距離を棒グラフで比較し、溶解性/懸濁安定性の結果をアノテーション]

### 9.9.2 共溶媒の役割

多くの吸入薬物はHFAプロペラント中で十分な溶解性を示さないため、エタノールなどの共溶媒が添加される。共溶媒のHSP値は、薬物とプロペラントの中間に位置することが望ましく、HSP空間における「橋渡し」効果により系全体の薬物溶解度を向上させる。

## 演習問題

### 演習9.1: イブプロフェンの溶媒スクリーニング

**問題:**

イブプロフェンのHSP値は $\delta_D = 17.4$, $\delta_P = 4.0$, $\delta_H = 9.6$ MPa$^{1/2}$、$R_0 = 10.2$ MPa$^{1/2}$ である。以下の5溶媒について $Ra$ と $RED$ を計算し、良溶媒/貧溶媒に分類せよ。

| 溶媒 | $\delta_D$ | $\delta_P$ | $\delta_H$ |
|------|-----------|-----------|-----------|
| エタノール | 15.8 | 8.8 | 19.4 |
| 酢酸エチル | 15.8 | 5.3 | 7.2 |
| 水 | 15.5 | 16.0 | 42.3 |
| ヘキサン | 14.9 | 0.0 | 0.0 |
| アセトン | 15.5 | 10.4 | 7.0 |

単位: MPa$^{1/2}$

---

**解法:**

各溶媒について $Ra$ を計算する。

$$
Ra = \sqrt{4(\delta_{D,ibu} - \delta_{D,sol})^2 + (\delta_{P,ibu} - \delta_{P,sol})^2 + (\delta_{H,ibu} - \delta_{H,sol})^2}
$$

**エタノール:**

$Ra = \sqrt{4(17.4 - 15.8)^2 + (4.0 - 8.8)^2 + (9.6 - 19.4)^2}$

$= \sqrt{4(1.6)^2 + (-4.8)^2 + (-9.8)^2}$

$= \sqrt{4 \times 2.56 + 23.04 + 96.04}$

$= \sqrt{10.24 + 23.04 + 96.04} = \sqrt{129.32} = 11.37$

$RED = 11.37 / 10.2 = 1.11$ → **貧溶媒（境界的）**

**酢酸エチル:**

$Ra = \sqrt{4(17.4 - 15.8)^2 + (4.0 - 5.3)^2 + (9.6 - 7.2)^2}$

$= \sqrt{4(1.6)^2 + (-1.3)^2 + (2.4)^2}$

$= \sqrt{10.24 + 1.69 + 5.76} = \sqrt{17.69} = 4.21$

$RED = 4.21 / 10.2 = 0.41$ → **良溶媒**

**水:**

$Ra = \sqrt{4(17.4 - 15.5)^2 + (4.0 - 16.0)^2 + (9.6 - 42.3)^2}$

$= \sqrt{4(1.9)^2 + (-12.0)^2 + (-32.7)^2}$

$= \sqrt{14.44 + 144.0 + 1069.29} = \sqrt{1227.73} = 35.04$

$RED = 35.04 / 10.2 = 3.44$ → **貧溶媒**

**ヘキサン:**

$Ra = \sqrt{4(17.4 - 14.9)^2 + (4.0 - 0.0)^2 + (9.6 - 0.0)^2}$

$= \sqrt{4(2.5)^2 + (4.0)^2 + (9.6)^2}$

$= \sqrt{25.0 + 16.0 + 92.16} = \sqrt{133.16} = 11.54$

$RED = 11.54 / 10.2 = 1.13$ → **貧溶媒（境界的）**

**アセトン:**

$Ra = \sqrt{4(17.4 - 15.5)^2 + (4.0 - 10.4)^2 + (9.6 - 7.0)^2}$

$= \sqrt{4(1.9)^2 + (-6.4)^2 + (2.6)^2}$

$= \sqrt{14.44 + 40.96 + 6.76} = \sqrt{62.16} = 7.88$

$RED = 7.88 / 10.2 = 0.77$ → **良溶媒**

---

**解答まとめ:**

| 溶媒 | $Ra$ (MPa$^{1/2}$) | RED | 分類 |
|------|:---:|:---:|------|
| エタノール | 11.37 | 1.11 | 貧溶媒（境界） |
| 酢酸エチル | 4.21 | 0.41 | 良溶媒 |
| 水 | 35.04 | 3.44 | 貧溶媒 |
| ヘキサン | 11.54 | 1.13 | 貧溶媒（境界） |
| アセトン | 7.88 | 0.77 | 良溶媒 |

> **考察:** イブプロフェンは $\delta_H$ が中程度の値を持つため、水素結合パラメータが極めて大きい水には溶解せず（RED = 3.44）、水素結合能を持たないヘキサンに対してもRED > 1となる。酢酸エチルやアセトンのように中程度の極性を持つ溶媒が良溶媒であり、イブプロフェンのBCS Class II（低水溶性）の分類と一致する。エタノールとヘキサンは境界的（RED ≈ 1.1）であり、温度変化により溶解性が大きく変動する領域に位置している。

### 演習9.2: カフェインの経皮促進剤選定

**問題:**

カフェインの経皮デリバリーにおいて、以下の条件で促進剤を選定せよ。

- カフェイン（薬物）: $\delta_D = 19.5$, $\delta_P = 11.4$, $\delta_H = 16.8$ MPa$^{1/2}$
- 角層脂質（皮膚）: $\delta_D = 16.0$, $\delta_P = 3.5$, $\delta_H = 6.0$ MPa$^{1/2}$

候補促進剤: オレイン酸、プロピレングリコール、DMSO、Azone、エタノール（HSP値は表9.3参照）

各促進剤について、$Ra_{enhancer-skin}$、$Ra_{drug-enhancer}$、$Ra_{drug-skin}$ を計算し、最適な促進剤を選定せよ。

---

**解法:**

まず、薬物-皮膚間のHSP距離を計算する。

$Ra_{drug-skin} = \sqrt{4(19.5-16.0)^2 + (11.4-3.5)^2 + (16.8-6.0)^2}$

$= \sqrt{4(3.5)^2 + (7.9)^2 + (10.8)^2}$

$= \sqrt{49.0 + 62.41 + 116.64} = \sqrt{228.05} = 15.10$ MPa$^{1/2}$

次に、各促進剤について $Ra_{enhancer-skin}$ と $Ra_{drug-enhancer}$ を計算する。

**オレイン酸** ($\delta_D = 16.2$, $\delta_P = 2.8$, $\delta_H = 5.5$):

$Ra_{enh-skin} = \sqrt{4(16.2-16.0)^2 + (2.8-3.5)^2 + (5.5-6.0)^2}$
$= \sqrt{4(0.04) + 0.49 + 0.25} = \sqrt{0.90} = 0.95$

$Ra_{drug-enh} = \sqrt{4(19.5-16.2)^2 + (11.4-2.8)^2 + (16.8-5.5)^2}$
$= \sqrt{4(10.89) + 73.96 + 127.69} = \sqrt{245.21} = 15.66$

**プロピレングリコール** ($\delta_D = 16.8$, $\delta_P = 10.4$, $\delta_H = 21.3$):

$Ra_{enh-skin} = \sqrt{4(16.8-16.0)^2 + (10.4-3.5)^2 + (21.3-6.0)^2}$
$= \sqrt{4(0.64) + 47.61 + 234.09} = \sqrt{284.26} = 16.86$

$Ra_{drug-enh} = \sqrt{4(19.5-16.8)^2 + (11.4-10.4)^2 + (16.8-21.3)^2}$
$= \sqrt{4(7.29) + 1.0 + 20.25} = \sqrt{50.41} = 7.10$

**DMSO** ($\delta_D = 18.4$, $\delta_P = 16.4$, $\delta_H = 10.2$):

$Ra_{enh-skin} = \sqrt{4(18.4-16.0)^2 + (16.4-3.5)^2 + (10.2-6.0)^2}$
$= \sqrt{4(5.76) + 166.41 + 17.64} = \sqrt{207.09} = 14.39$

$Ra_{drug-enh} = \sqrt{4(19.5-18.4)^2 + (11.4-16.4)^2 + (16.8-10.2)^2}$
$= \sqrt{4(1.21) + 25.0 + 43.56} = \sqrt{73.40} = 8.57$

**Azone** ($\delta_D = 16.0$, $\delta_P = 3.2$, $\delta_H = 4.8$):

$Ra_{enh-skin} = \sqrt{4(16.0-16.0)^2 + (3.2-3.5)^2 + (4.8-6.0)^2}$
$= \sqrt{0 + 0.09 + 1.44} = \sqrt{1.53} = 1.24$

$Ra_{drug-enh} = \sqrt{4(19.5-16.0)^2 + (11.4-3.2)^2 + (16.8-4.8)^2}$
$= \sqrt{49.0 + 67.24 + 144.0} = \sqrt{260.24} = 16.13$

**エタノール** ($\delta_D = 15.8$, $\delta_P = 8.8$, $\delta_H = 19.4$):

$Ra_{enh-skin} = \sqrt{4(15.8-16.0)^2 + (8.8-3.5)^2 + (19.4-6.0)^2}$
$= \sqrt{4(0.04) + 28.09 + 179.56} = \sqrt{207.81} = 14.42$

$Ra_{drug-enh} = \sqrt{4(19.5-15.8)^2 + (11.4-8.8)^2 + (16.8-19.4)^2}$
$= \sqrt{4(13.69) + 6.76 + 6.76} = \sqrt{68.28} = 8.26$

---

**解答まとめ:**

| 促進剤 | $Ra_{enh-skin}$ | $Ra_{drug-enh}$ | バリア親和性 | 薬物親和性 |
|--------|:---:|:---:|:---:|:---:|
| オレイン酸 | 0.95 | 15.66 | 極めて高い | 低い |
| プロピレングリコール | 16.86 | 7.10 | 低い | 中程度 |
| DMSO | 14.39 | 8.57 | 低い | 中程度 |
| Azone | 1.24 | 16.13 | 極めて高い | 低い |
| エタノール | 14.42 | 8.26 | 低い | 中程度 |

> **考察と推奨:** カフェインは極性・水素結合パラメータが大きく、角層脂質との $Ra$ が15.10と非常に大きい。単一促進剤では理想的な候補が存在しない。オレイン酸とAzoneは角層脂質への親和性が極めて高いが、カフェインとの親和性は低い。一方、プロピレングリコールやエタノールはカフェインとの親和性が比較的高いが、角層脂質への浸透性は低い。
>
> **推奨戦略:** バリア親和型促進剤（オレイン酸またはAzone）と薬物溶媒和型促進剤（プロピレングリコールまたはエタノール）の二元系の使用が最適である。例えば、オレイン酸 + プロピレングリコールの組み合わせは、角層脂質構造の流動化（オレイン酸）とカフェインの角層内溶解度向上（プロピレングリコール）を同時に実現できる。

### 演習9.3: カルバマゼピンの共結晶スクリーニング

**問題:**

カルバマゼピン（$\delta_D = 19.4$, $\delta_P = 8.8$, $\delta_H = 8.5$ MPa$^{1/2}$）に対して、以下の10種の候補コフォーマーのHSP距離を計算し、共結晶形成の可能性を評価せよ。

| コフォーマー | $\delta_D$ | $\delta_P$ | $\delta_H$ |
|------------|-----------|-----------|-----------|
| サッカリン | 20.8 | 12.5 | 10.2 |
| ニコチンアミド | 18.8 | 14.2 | 12.5 |
| グルタル酸 | 17.2 | 7.5 | 13.5 |
| コハク酸 | 17.5 | 8.2 | 12.8 |
| 安息香酸 | 19.5 | 7.5 | 9.8 |
| マレイン酸 | 17.0 | 10.5 | 15.0 |
| マンニトール | 17.0 | 10.2 | 24.5 |
| 尿素 | 17.5 | 15.0 | 18.5 |
| フマル酸 | 18.0 | 11.5 | 14.8 |
| ステアリン酸 | 16.2 | 2.5 | 3.0 |

単位: MPa$^{1/2}$

---

**解法:**

各コフォーマーについて $\Delta HSP$ を計算する（等重みモデル）。

$$
\Delta HSP = \sqrt{(\delta_{D,CBZ} - \delta_{D,cof})^2 + (\delta_{P,CBZ} - \delta_{P,cof})^2 + (\delta_{H,CBZ} - \delta_{H,cof})^2}
$$

**サッカリン:**
$\Delta HSP = \sqrt{(19.4-20.8)^2 + (8.8-12.5)^2 + (8.5-10.2)^2}$
$= \sqrt{1.96 + 13.69 + 2.89} = \sqrt{18.54} = 4.31$

**ニコチンアミド:**
$\Delta HSP = \sqrt{(0.6)^2 + (-5.4)^2 + (-4.0)^2} = \sqrt{0.36 + 29.16 + 16.0} = \sqrt{45.52} = 6.75$

**グルタル酸:**
$\Delta HSP = \sqrt{(2.2)^2 + (1.3)^2 + (-5.0)^2} = \sqrt{4.84 + 1.69 + 25.0} = \sqrt{31.53} = 5.62$

**コハク酸:**
$\Delta HSP = \sqrt{(1.9)^2 + (0.6)^2 + (-4.3)^2} = \sqrt{3.61 + 0.36 + 18.49} = \sqrt{22.46} = 4.74$

**安息香酸:**
$\Delta HSP = \sqrt{(-0.1)^2 + (1.3)^2 + (-1.3)^2} = \sqrt{0.01 + 1.69 + 1.69} = \sqrt{3.39} = 1.84$

**マレイン酸:**
$\Delta HSP = \sqrt{(2.4)^2 + (-1.7)^2 + (-6.5)^2} = \sqrt{5.76 + 2.89 + 42.25} = \sqrt{50.90} = 7.13$

**マンニトール:**
$\Delta HSP = \sqrt{(2.4)^2 + (-1.4)^2 + (-16.0)^2} = \sqrt{5.76 + 1.96 + 256.0} = \sqrt{263.72} = 16.24$

**尿素:**
$\Delta HSP = \sqrt{(1.9)^2 + (-6.2)^2 + (-10.0)^2} = \sqrt{3.61 + 38.44 + 100.0} = \sqrt{142.05} = 11.92$

**フマル酸:**
$\Delta HSP = \sqrt{(1.4)^2 + (-2.7)^2 + (-6.3)^2} = \sqrt{1.96 + 7.29 + 39.69} = \sqrt{48.94} = 7.00$

**ステアリン酸:**
$\Delta HSP = \sqrt{(3.2)^2 + (6.3)^2 + (5.5)^2} = \sqrt{10.24 + 39.69 + 30.25} = \sqrt{80.18} = 8.95$

---

**解答まとめ:**

| コフォーマー | $\Delta HSP$ (MPa$^{1/2}$) | 共結晶形成予測 | 実験結果 |
|------------|:---:|:---:|:---:|

| コハク酸 | 4.74 | 可能（境界的） | Yes (CSD) |
| グルタル酸 | 5.62 | 高い（最適範囲） | Yes (CSD) |
| ニコチンアミド | 6.75 | 高い（最適範囲） | Yes (CSD) |
| フマル酸 | 7.00 | 高い（最適範囲） | Yes |
| マレイン酸 | 7.13 | 高い（最適範囲） | Yes (CSD) |
| ステアリン酸 | 8.95 | 中程度（境界的） | No |

| マンニトール | 16.24 | 低い（距離過大） | No |

> **考察:** $\Delta HSP$ 基準による共結晶形成予測は、カルバマゼピンの既知の共結晶データと概ね良好に一致する。特に5--10 MPa$^{1/2}$ の範囲にあるグルタル酸、ニコチンアミド、フマル酸、マレイン酸は実際に共結晶形成が確認されている。安息香酸は $\Delta HSP = 1.84$ と距離が極めて小さく、固溶体を形成する傾向が予測され、実験結果とも一致する。ステアリン酸は $\Delta HSP = 8.95$ で境界的であるが、長鎖脂肪酸のパッキング因子がHSPでは捉えられない立体的制約を導入するため、共結晶が形成されないと考えられる。**HSP距離は有効な予備スクリーニングツールであるが、分子形状・結晶パッキング効率などのHSP外の因子を考慮した最終的な実験的検証は不可欠である。**

### 演習9.4: パラセタモールの多形リスク評価

## 章末問題

1. RED分類システムが医薬用溶媒中の薬物溶解性予測にどのように適用されるかを説明せよ。特に多官能基を持つ薬物について、単一の球で溶解挙動を表現することの限界を論ぜよ。

2. 経皮浸透促進剤選定における三成分HSPフレームワーク（薬物-皮膚-促進剤）を説明せよ。促進剤が薬物と皮膚脂質バリア間のHSP距離をどのように修飾するか、また、HSP以外に経皮デリバリーに影響する実用的因子は何かを論ぜよ。

3. REDゾーン0.5--1.5が結晶化時に多形・溶媒和物リスクの増大を呈する理由を説明せよ。HSP解析を用いて望ましくない結晶形のリスクを最小化する結晶化プロセス開発戦略を提案せよ。

4. 共結晶スクリーニングにおけるHSPアプローチを、他の計算手法（COSMO-RS、分子相補性など）と比較せよ。HSP距離を用いたAPI-コフォーマー共結晶形成予測の利点と限界は何か。

5. 薬物と高分子キャリアのHSPマッチングがナノ粒子薬物担持効率と放出動態にどのように影響するかを論ぜよ。$\chi$ パラメータが薬物担持ナノ粒子の長期安定性予測においてどのような役割を果たすか。

## ソフトウェア実習

本章の内容を実践的に理解するため、以下のソフトウェア実習を行う。

### 実習手順

1. **薬物溶解性（Drug Solubility）タブ**を開き、イブプロフェンのHSP値（$\delta_D = 17.4$, $\delta_P = 4.0$, $\delta_H = 9.6$）と $R_0 = 10.2$ を入力する。
2. 表示される溶媒ランキングリスト（RED値の昇順）を確認し、各溶媒のカラーコード分類（良溶媒/部分溶媒/貧溶媒）を検証する。既知の実験的溶解度データとの一致を確認する。
3. **分散剤選定（Dispersant Selection）タブ**に切り替え、経皮デリバリーの三成分システムを構成する。薬物HSP（カフェイン: $\delta_D = 19.5$, $\delta_P = 11.4$, $\delta_H = 16.8$）と皮膚HSP（$\delta_D = 16.0$, $\delta_P = 3.5$, $\delta_H = 6.0$）を入力し、候補促進剤のHSP値を登録する。
4. 促進剤候補の予測経皮促進ランキングを評価し、最適な促進剤または二元系組み合わせを選定する。
5. **共結晶スクリーニング（Cocrystal Screening）タブ**を開き、カルバマゼピンのHSP値と10種のコフォーマー候補のHSP値を入力する。
6. API-コフォーマーHSP距離と共結晶形成尤度スコアを確認し、有望な共結晶対を同定する。

### 期待される出力

- **薬物溶解性タブ**: 各溶媒のRED値（昇順ランキング）、カラーコード分類（良溶媒 = 緑、部分溶媒 = 黄、貧溶媒 = 赤）
- **分散剤選定タブ**: 促進剤候補の予測経皮促進効果ランキング（三成分HSP距離スコア）
- **共結晶スクリーニングタブ**: API-コフォーマーHSP距離と共結晶形成尤度スコア

## コラム: 製剤科学者の視点から — HSPが変える薬づくり

> 「薬の開発現場では、候補化合物が見つかっても『溶けない』という壁にぶつかることが少なくありません。現在、新薬候補の約40%がBCS Class IIに分類される難水溶性化合物です。これらの薬物をいかにして体内で溶かし、必要な場所に届けるか -- これが製剤科学者の中心的な課題です。
>
> 従来の製剤開発は、文字通りの試行錯誤でした。数十種の溶媒、数十種の高分子、数十種の賦形剤を組み合わせて最適な処方を見つけ出す。時間もコストも莫大にかかります。HSPは、この網羅的スクリーニングに代わる合理的な出発点を提供してくれます。
>
> 共結晶の研究では、100種のコフォーマーを全部試す代わりに、HSP距離で絞り込んだ20種を優先的に検討する。ナノ粒子の設計では、薬物との $\chi$ パラメータが最小の高分子から実験を始める。経皮パッチの開発では、三成分HSP解析で促進剤候補を5種に絞ってから膜透過実験に進む。このように、HSPは『何から試すべきか』の指針を与えてくれるのです。
>
> もちろん、HSPには限界もあります。生体内での代謝や分布、結晶の立体構造、膜輸送体の関与など、HSPでは捉えきれない因子は数多い。しかし、熱力学的な第一近似として、HSPは製剤設計のあらゆる局面で有用な羅針盤です。」
