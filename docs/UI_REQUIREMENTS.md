# UI画面要件定義書

> Generated: 2026-03-22 | 対象: 93タブ (6カテゴリ)

## 1. 共通UIパターン

### 1.1 入力パターン

| パターン | 説明 | 使用例 |
|---------|------|-------|
| **単一セレクタ** | PartsGroupSelector のみ | ESC, 3D印刷平滑化 |
| **二重セレクタ** | PartsGroup + Solvent | 溶解性評価, 膨潤度予測 |
| **カスケード** | Group → Part ドロップダウン | 可塑剤選定, ナノ粒子分散 |
| **マルチ選択** | チェックボックスリスト | ブレンド最適化, リサイクル相溶性 |
| **カテゴリフィルタ** | カテゴリで候補を絞込 | ナノ粒子分散, 分散剤選定 |
| **モード切替** | 2-3モードの評価方式切替 | 接触角推定, 薬物溶解性, 分散剤選定 |
| **HSP手動入力** | δD/δP/δH 数値入力 | インク-基材密着, 構造接着設計 |
| **動的リスト** | 行の追加/削除 | コポリマーHSP推定, 多層コーティング |
| **制約条件** | チェックボックスで有効化 | ナノ粒子分散, 多目的選定 |

### 1.2 結果表示パターン

| パターン | 説明 | 使用例 |
|---------|------|-------|

| **バッジ分類** | 色分けレベル表示 | 全evaluation/selection |

## 4. UIパターン別テストテンプレート

### 4.1 screening型テスト (40+ タブ)
```
1. タブ遷移 → タイトル表示確認
2. セレクタにシードデータ表示確認
3. 材料選択 → 情報カード表示確認
4. 実行ボタンクリック → loading → 結果テーブル表示
5. 統計サマリー表示確認
6. Badge表示確認 (>=1件)
7. ソートヘッダークリック → ▲▼切替
8. CSV出力ボタン表示確認
```

### 4.2 pairwise型テスト (15+ タブ)
```
1. タブ遷移 → タイトル表示確認
2. 入力フィールド全て表示確認
3. 値入力 → 実行ボタン有効化
4. 実行 → 結果カード表示 (Wa/Ra/レベル)
5. Badge表示確認
```

### 4.3 optimization型テスト (10+ タブ)
```
1. タブ遷移 → タイトル表示確認
2. ターゲットHSP入力 → 候補選択
3. 実行 → 最適化結果テーブル表示
4. 結果にブレンド組成情報含む確認
```

### 4.4 analysis/tool型テスト (16 タブ)
```
1. タブ遷移 → タイトル表示確認
2. 入力コントロール表示確認
3. (該当する場合) 実行 → 結果表示
```
