# HSPによる膨潤度予測 調査レポート

調査日: 2026-03-15

---

## 1. エグゼクティブサマリー

エラストマー・ゴム材料の溶媒による膨潤現象を、Hansen溶解度パラメータ（HSP）を用いて予測する手法を調査した。膨潤はFlory-Huggins相互作用パラメータχに支配されるが、HSP距離（Ra）および相対エネルギー差（RED = Ra/R₀）がχの良好な代替指標となることを確認した。特に3次元HSP空間ではREDがχよりも優れた予測因子であり、RED < 1の場合に膨潤リスクが高い。

**結論:** REDベースの4段階分類（severe/high/moderate/low）により、実験（ASTM D471）なしで膨潤リスクの迅速スクリーニングが可能である。ただし本手法はエラストマー・ゴム材料のみに適用可能であり、剛直ポリマーには適用できない。

---

## 2. 理論的背景

### 2.1 膨潤現象のメカニズム

エラストマーの膨潤は以下の段階で進行する：

```
1. 溶媒分子がエラストマー表面に接触
2. 溶媒分子がポリマー鎖間に拡散（Fickian拡散）
3. ポリマー鎖が緩和・膨張（鎖のコンフォメーション変化）
4. 溶媒浸透の膨張圧と架橋による弾性収縮力が平衡に達する
5. 平衡膨潤状態の確立
```

平衡膨潤度は、ポリマー-溶媒間の熱力学的親和性と架橋密度に支配される。

### 2.2 Flory-Huggins理論

Flory-Huggins相互作用パラメータχは、ポリマー-溶媒間の混合の自由エネルギーを表す：

```
ΔG_mix / (n_s · kT) = ln(1-φ_p) + (1 - 1/N)·φ_p + χ·φ_p²
```

ここで：
- `n_s` = 溶媒分子数
- `φ_p` = ポリマーの体積分率
- `N` = ポリマーの重合度
- `χ` = Flory-Huggins相互作用パラメータ

χパラメータは溶解度パラメータから推定できる：

```
χ = V_m · (δ₁ - δ₂)² / (RT)
```

ここで：
- `V_m` = 溶媒のモル体積 (cm³/mol)
- `δ₁, δ₂` = 溶媒、ポリマーの溶解度パラメータ (MPa^1/2)
- `R` = 気体定数 (8.314 J/(mol·K))
- `T` = 温度 (K)

### 2.3 χと膨潤度の関係

Flory-Rehner式により、平衡膨潤度からχを決定できる：

```
ln(1 - v₂) + v₂ + χ·v₂² = -V_m · ν_e · (v₂^(1/3) - v₂/2)
```

ここで：
- `v₂` = 平衡膨潤時のポリマー体積分率
- `ν_e` = 有効架橋密度 (mol/cm³)

**χの目安:**
- χ < 0.5: 良好な溶媒（大きな膨潤）
- χ ≈ 0.5: θ溶媒（臨界条件）
- χ > 0.5: 貧溶媒（膨潤なし〜軽微）

### 2.4 HSP距離とFlory-Huggins χの関係

1次元の溶解度パラメータδでは：

```
χ = V_m · (δ₁ - δ₂)² / (RT)
```

3次元HSP空間ではRa²を使用：

```
χ_HSP ≈ V_m · Ra² / (4 · RT)
```

ここでRa²の係数4は、Hansen距離における分散力項の重み `4·ΔδD²` に由来する補正である。ただし、この変換にはモル体積V_mが必要であり、全溶剤に対してV_mデータが利用可能とは限らない。

## 3. REDによる膨潤予測

### 3.1 なぜREDがχより優れた予測因子なのか

χパラメータには以下の問題がある：

1. **1次元性**: 古典的χは全溶解度パラメータδの差のみを使用し、分散力・極性力・水素結合の個別寄与を区別しない
2. **モル体積依存**: V_mが必要で、データの欠損や温度依存性が問題
3. **大きな分子で過大評価**: V_mが大きい溶媒ではχが過大に計算される

REDの利点：

| 比較項目 | χ (Flory-Huggins) | RED (Hansen) |
|---------|-------------------|-------------|
| HSP次元 | 1次元（δ全体） | 3次元（δD, δP, δH個別） |
| 追加パラメータ | V_m, T が必要 | R₀のみ（材料固有） |
| 温度依存性 | 直接含む | 間接的のみ |
| 閾値 | 0.5（理論的だが厳密でない） | 1.0（Hansen球の境界） |
| 分散力の重み | なし | 4倍の重み（Ra²式） |
| 実験データとの整合性 | 中程度 | 良好 |

**結論:** REDは3次元HSP情報をフルに活用し、追加パラメータなしで膨潤リスクを直感的に評価できる。

### 3.2 REDベースの膨潤リスク分類

Hansen球の概念に基づき、以下の4段階分類を設計した：

| レベル | 名称 | RED範囲 | 物理的意味 |
|--------|------|---------|-----------|
| Severe（重度） | 深刻な膨潤 | RED < 0.5 | Hansen球の深部。溶媒がポリマーを強く膨潤させる |
| High（高度） | 大きな膨潤 | 0.5 ≤ RED < 0.8 | Hansen球内部。明確な膨潤が予想される |
| Moderate（中度） | 中程度の膨潤 | 0.8 ≤ RED < 1.0 | Hansen球の境界付近。限定的な膨潤の可能性 |
| Low（低度） | 軽微な膨潤 | 1.0 ≤ RED < 1.5 | Hansen球外だが近い。軽微な膨潤の可能性あり |
| None（なし） | 膨潤なし | RED ≥ 1.5 | 明確にHansen球外。膨潤リスクは極めて低い |

### 3.3 閾値の根拠

- **severeMax = 0.5**: Hansen球の中心に近く、ポリマーと溶媒のHSPがほぼ一致。最大膨潤が予想される領域
- **highMax = 0.8**: Hansen球の内側で十分な親和性がある領域
- **moderateMax = 1.0**: Hansen球の境界。これを超えると基本的に非相溶
- **lowMax = 1.5**: Hansen球外だが、僅かな浸透が起こりうる遷移領域。架橋密度が低い場合などに軽微な膨潤が観察される

## 4. エラストマーの膨潤と架橋密度

### 4.1 架橋密度の影響

膨潤度は架橋密度に強く依存する：

```
Q_eq ∝ ν_e^(-3/5)
```

ここで `Q_eq` = 平衡膨潤比（膨潤後体積/初期体積）、`ν_e` = 有効架橋密度。

架橋密度が高いほど鎖の膨張が制限され、同じχ値でも膨潤度は小さくなる。HSPベースの予測では架橋密度を直接考慮できないため、これは主要な限界事項である。

### 4.2 代表的なエラストマーのHSP値と膨潤特性

| エラストマー | δD | δP | δH | R₀ | 膨潤に弱い溶媒群 |
|-------------|----|----|-----|-----|---------------|
| 天然ゴム（NR） | 17.4 | 3.1 | 4.1 | 10.6 | 芳香族炭化水素、塩素系溶剤 |
| ニトリルゴム（NBR） | 18.6 | 12.2 | 4.7 | 9.1 | ケトン、エステル |
| EPDM | 17.2 | 2.0 | 2.8 | 8.2 | 脂肪族炭化水素 |
| フッ素ゴム（FKM） | 17.6 | 11.3 | 3.5 | 7.5 | 低分子ケトン、エステル |
| シリコーンゴム（VMQ） | 15.9 | 0.1 | 4.7 | 12.3 | 非極性溶媒全般 |

### 4.3 ASTM D471との比較

ASTM D471は「ゴムの液体への浸漬による影響を評価する標準試験法」である。

**試験方法:**
1. ゴム試験片を所定の液体に浸漬（通常70°Cまたは100°C、70時間）
2. 体積変化率（ΔV%）、質量変化率（ΔM%）、硬度変化（ΔH）を測定
3. 引張強度・伸びの変化も評価

**HSP予測との対応:**

| ASTM D471 ΔV% | HSP RED範囲（概算） | 対応 |
|----------------|-------------------|------|
| > 100% | RED < 0.3 | 壊滅的膨潤 |
| 50-100% | 0.3-0.6 | 重度 |
| 20-50% | 0.6-0.9 | 高度-中度 |
| 5-20% | 0.9-1.2 | 中度-低度 |
| < 5% | RED > 1.2 | 低度-なし |

**注意:** この対応は概算であり、架橋密度・温度・浸漬時間に大きく依存する。

## 5. 膨潤予測の適用条件

### 5.1 materialType チェック

本手法は**エラストマー・ゴム材料のみ**に適用可能である。

```
適用可能: エラストマー、ゴム（架橋ポリマー）
適用不可: 熱可塑性プラスチック、熱硬化性樹脂（剛直ポリマー）
```

**理由:**

1. **膨潤の機構が異なる**: 剛直ポリマーは溶媒を吸収しても弾性的に膨潤しない。溶解（dissolution）またはクレイジング（crazing）が起こる
2. **Flory-Rehner理論の前提**: 柔軟な架橋鎖がランダムコイルのように振舞うことを仮定。ガラス状ポリマーではこの仮定が成立しない
3. **架橋構造**: エラストマーは化学架橋により3次元ネットワークを形成し、溶解せずに膨潤のみ起こる。非架橋ポリマーは完全に溶解しうる

### 5.2 温度の影響

膨潤は温度上昇に伴い増大する：

- 拡散速度 D ∝ exp(-E_a/RT) の増加
- ポリマー鎖の運動性増大
- χパラメータの変化

実用上、室温（25°C）でのHSPデータに基づく予測が標準であり、高温環境（例: エンジン周辺部品のオイル接触）では保守的な評価が必要。

## 6. 実装における設計判断

### 6.1 REDベース vs χベースの選択

| 判断基準 | 選択 | 理由 |
|---------|------|------|
| 予測精度 | RED | 3次元HSP情報をフル活用 |
| 追加データ不要 | RED | V_mが不要 |
| 直感性 | RED | Hansen球という確立された概念 |
| 閾値の明確性 | RED | 1.0 = Hansen球境界 |

### 6.2 分類閾値の設定

分類閾値は `SwellingThresholds` として外部パラメータ化し、ユーザーが調整可能とする：

```
interface SwellingThresholds {
  severeMax: number;   // default: 0.5
  highMax: number;     // default: 0.8
  moderateMax: number; // default: 1.0
  lowMax: number;      // default: 1.5
}
```

### 6.3 materialType バリデーション

入力されたPart（材料）のタイプが `elastomer` または `rubber` であることを検証し、それ以外の場合はエラーまたは警告を返す設計とする。

## 7. 実用的計算例

### 例: ニトリルゴム（NBR）O-リングの溶媒暴露評価

NBR: δD=18.6, δP=12.2, δH=4.7, R₀=9.1

| 溶媒 | δD | δP | δH | Ra | RED | 膨潤リスク |
|------|----|----|-----|------|------|----------|
| トルエン | 18.0 | 1.4 | 2.0 | 11.1 | 1.22 | Low |
| アセトン | 15.5 | 10.4 | 7.0 | 7.0 | 0.77 | High |
| MEK | 16.0 | 9.0 | 5.1 | 6.2 | 0.68 | High |
| ヘキサン | 14.9 | 0.0 | 0.0 | 14.6 | 1.60 | None |
| エタノール | 15.8 | 8.8 | 19.4 | 16.3 | 1.79 | None |
| DMF | 17.4 | 13.7 | 11.3 | 7.4 | 0.81 | Moderate |
| ジクロロメタン | 17.0 | 7.3 | 7.1 | 6.2 | 0.68 | High |

**解釈:**
- NBRはケトン類（アセトン、MEK）やハロゲン化溶剤（ジクロロメタン）に対して膨潤リスクが高い
- 脂肪族炭化水素（ヘキサン）やアルコール（エタノール）には耐性がある
- これはNBRの一般的に知られた耐薬品性と一致する

## 8. 限界事項と注意点

### 8.1 架橋密度の非考慮

HSPベースの予測では架橋密度を直接考慮できない。同じNBRでも、架橋密度が異なれば実際の膨潤度は大きく異なる。

### 8.2 フィラーの影響

カーボンブラック、シリカなどのフィラーは：
- ゴムの有効架橋密度を増加させ膨潤を抑制
- フィラー-溶媒相互作用は考慮されない
- 高充填系では膨潤度が30-50%減少することがある

### 8.3 化学的攻撃

強酸、強塩基、強酸化剤によるポリマー鎖の化学的分解は膨潤とは異なる現象であり、HSPでは予測できない。

### 8.4 混合溶媒環境

実環境では複数の溶媒に同時に暴露されることが多い。混合溶媒のHSPは線形混合則で推定できるが、選択的吸収（特定成分のみが優先的に浸透）は考慮されない。

### 8.5 経時変化

長期間の溶媒暴露では：
- 可塑剤の抽出
- 架橋の切断
- 老化・劣化

これらの不可逆変化はHSPモデルの範囲外である。

## 10. 実装判断のまとめ

| 判断項目 | 決定 | 理由 |
|---------|------|------|
| 予測指標 | RED（Ra/R₀） | 3次元HSP対応、追加パラメータ不要 |
| 分類段階 | 5段階（Severe/High/Moderate/Low/None） | 実務で必要な判断粒度 |
| 閾値 | severeMax=0.5, highMax=0.8, moderateMax=1.0, lowMax=1.5 | Hansen球の物理的意味に基づく |
| materialType制限 | エラストマー/ゴムのみ | 理論的前提がエラストマーに限定 |
| 架橋密度 | 非考慮（限界事項として明記） | データ不足、材料バリエーションが大きい |
| 温度 | 25°C基準（注記あり） | HSPデータベースの標準条件 |
