# HSPによる薬物溶解性予測 調査レポート

調査日: 2026-03-15

---

## 1. エグゼクティブサマリー

Hansen溶解度パラメータ（HSP）を用いた薬物（API: Active Pharmaceutical Ingredient）の溶解性予測手法を調査した。HSP距離（Ra）および相対エネルギー差（RED）は、薬物-溶媒間の混和性の指標として有効であり、BCS分類における難溶性薬物のスクリーニングや処方設計に活用できる。logP、Yalkowsky式、COSMO-RSなどの既存手法と比較し、HSPの利点（実験データ不要、迅速なスクリーニング、構造的洞察）と限界（結晶充填効果、イオン化/pH効果が反映されない）を整理した。

**結論:** HSPは薬物の溶解性スクリーニングおよび溶媒選定の初期段階で有効なツールであるが、結晶多形・イオン化状態などの影響を考慮できないため、最終的な処方決定には実験的検証が不可欠である。

---

## 2. 理論的背景

### 2.1 薬物溶解性の重要性

薬物の経口バイオアベイラビリティは溶解性と膜透過性に支配される。BCS（Biopharmaceutics Classification System）分類：

| BCSクラス | 溶解性 | 膜透過性 | 代表的薬物 |
|----------|--------|---------|-----------|
| I | 高い | 高い | アセトアミノフェン、メトプロロール |
| II | 低い | 高い | イブプロフェン、カルバマゼピン |
| III | 高い | 低い | アテノロール、メトホルミン |
| IV | 低い | 低い | フロセミド、タキソール |

BCS Class IIおよびIVの薬物では溶解性の向上が製剤設計の鍵となる。HSPは適切な溶媒・賦形剤の選定に有用である。

### 2.2 HSPによる溶解性予測の原理

「似たもの同士が溶ける（Like dissolves like）」の原理に基づき、薬物と溶媒のHSP値が近いほど混和性が高い。

```
Ra² = 4·(δD_drug − δD_solv)² + (δP_drug − δP_solv)² + (δH_drug − δH_solv)²
RED = Ra / R₀
```

| RED値 | 溶解性予測 | 製剤への意味 |
|-------|-----------|-------------|
| RED < 0.5 | 高溶解性 | 高い薬物充填量（drug loading）が期待 |
| 0.5 ≤ RED < 1.0 | 溶解する | 実用的な溶解性 |
| 1.0 ≤ RED < 1.5 | 限定的 | 低い溶解性、共溶媒等の工夫が必要 |
| RED ≥ 1.5 | 不溶 | 当該溶媒での溶解は困難 |

### 2.3 HSP値の決定：薬物分子の場合

薬物のHSP値は以下の手法で決定される：

1. **溶解試験法**: 多数の溶媒で溶解性を評価し、Hansen球を構築（最も信頼性が高い）
2. **グループ寄与法**: Van Krevelen法やHoy法で分子構造からHSP値を推算
3. **逆ガスクロマトグラフィー（IGC）**: 固体のHSP値を実験的に決定
4. **分子シミュレーション**: MD計算による凝集エネルギー密度の算出

## 3. 他の溶解性予測手法との比較

### 3.1 logP（分配係数）

オクタノール/水分配係数。疎水性の指標。

```
logP = log([薬物]_octanol / [薬物]_water)
```

**利点:** 広く普及、データベースが豊富、計算ツール多数
**限界:** 1次元指標であり、溶媒特異性を区別できない

### 3.2 Yalkowsky式（一般溶解度方程式）

```
log S_w = 0.5 - 0.01 · (T_m - 25) - log K_ow
```

ここで S_w = 水溶解度（mol/L）、T_m = 融点（°C）、K_ow = オクタノール/水分配係数。

**利点:** 融点とlogPのみで水溶解度を推算可能
**限界:** 水溶解度のみ、有機溶媒系には適用不可

### 3.3 COSMO-RS

量子化学計算に基づくスクリーニングモデル。

```
σ-profile → σ-potential → 化学ポテンシャル → 溶解度
```

**利点:** 高精度、分子レベルの洞察
**限界:** 計算コストが高い、専用ソフトウェアが必要

### 3.4 手法比較まとめ

| 手法 | 入力 | 対象溶媒 | 精度 | 速度 | コスト |
|------|------|---------|------|------|--------|
| **HSP/RED** | δD, δP, δH, R₀ | 任意の溶媒 | 定性〜半定量 | 即時 | 無料 |
| logP | 分子構造 | 水/オクタノール系のみ | 半定量 | 即時 | 無料 |
| Yalkowsky | T_m, logP | 水のみ | 半定量 | 即時 | 無料 |
| COSMO-RS | 分子構造（3D） | 任意の溶媒 | 定量的 | 数時間 | 高い |
| 実験 | 試料 | 測定した溶媒 | 定量的 | 数日〜数週 | 中〜高 |

**HSPの優位性:**
1. 任意の溶媒に対して予測可能（水系に限定されない）
2. 実験データなしで迅速スクリーニングが可能
3. 溶媒選定の構造的洞察を提供（δD/δP/δHの寄与を分離）

## 4. 代表的薬物のHSP値と検証

### 4.1 シードデータの検証

HSPiPデータベースおよび文献値を基に、代表的薬物のHSP値を検証した。

#### アセトアミノフェン（パラセタモール）

```
δD = 18.1, δP = 10.2, δH = 16.2 (MPa^1/2)
R₀ ≈ 11.0
分子量: 151.2 g/mol
BCS Class I（高溶解・高透過）
```

水に対する溶解性検証：

```
水: δD=15.5, δP=16.0, δH=42.3
Ra = √[4×(18.1-15.5)² + (10.2-16.0)² + (16.2-42.3)²]
   = √[4×6.76 + 33.64 + 680.41]
   = √[741.09] = 27.2
RED = 27.2 / 11.0 = 2.47 → 不溶予測
```

実際のアセトアミノフェンの水溶解度は約14 mg/mL（25°C）であり、やや溶ける。水のδH=42.3が大きく影響し、RED値が実際の溶解性を過小評価している。これは水素結合の特殊性（水の自己会合）によるものと考えられる。

#### イブプロフェン

```
δD = 17.6, δP = 3.7, δH = 7.0 (MPa^1/2)
R₀ ≈ 10.0
分子量: 206.3 g/mol
BCS Class II（低溶解・高透過）
```

エタノールに対する溶解性検証：

```
エタノール: δD=15.8, δP=8.8, δH=19.4
Ra = √[4×(17.6-15.8)² + (3.7-8.8)² + (7.0-19.4)²]
   = √[4×3.24 + 26.01 + 153.76]
   = √[192.73] = 13.9
RED = 13.9 / 10.0 = 1.39 → 限定的溶解
```

実際のイブプロフェンはエタノールに良く溶ける（>200 mg/mL）。RED値がやや過大評価だが、水への溶解性（~0.05 mg/mL）と比較すると方向性は正しい。

#### カフェイン

```
δD = 19.5, δP = 13.3, δH = 11.5 (MPa^1/2)
R₀ ≈ 10.0
分子量: 194.2 g/mol
BCS Class I
```

水に対する溶解性検証：

```
水: δD=15.5, δP=16.0, δH=42.3
Ra = √[4×(19.5-15.5)² + (13.3-16.0)² + (11.5-42.3)²]
   = √[64.0 + 7.29 + 948.64]
   = √[1019.93] = 31.9
RED = 31.9 / 10.0 = 3.19 → 不溶予測
```

実際のカフェインの水溶解度は約22 mg/mL（25°C）であり、かなり溶ける。水系での乖離は前述と同様の原因による。

### 4.2 水系における限界の考察

水は特殊な溶媒であり、HSPによる溶解性予測が最も困難な系の一つである：

1. **水の自己会合**: 水分子は強い水素結合ネットワークを形成し、δH=42.3と異常に大きい
2. **疎水性水和**: 非極性分子の周囲で水構造が再編成される効果はHSPに含まれない
3. **イオン性相互作用**: 酸性・塩基性薬物のイオン化はHSPモデルの範囲外

**有機溶媒系ではHSPの予測精度は大幅に向上する。** 水系のスクリーニングよりも、有機溶媒選定（結晶化溶媒、抽出溶媒等）への応用が推奨される。

## 5. 薬物-賦形剤適合性スクリーニング

### 5.1 用途

固体分散体（solid dispersion）設計において、APIとポリマー賦形剤の混和性が重要：

```
REDが小さい → API-ポリマー親和性が高い → 安定な非晶質分散体
REDが大きい → API-ポリマー親和性が低い → 相分離・再結晶リスク
```

### 5.2 代表的な賦形剤ポリマーのHSP値

| 賦形剤 | δD | δP | δH | 主な用途 |
|--------|----|----|-----|---------|
| PVP（ポビドン） | 17.4 | 11.6 | 21.6 | 固体分散体、結合剤 |
| HPMC | 18.0 | 14.2 | 16.8 | 徐放性マトリクス |
| PEG 6000 | 17.0 | 8.7 | 11.8 | 固体分散体 |
| Eudragit L100 | 18.2 | 9.5 | 9.0 | 腸溶性コーティング |
| エチルセルロース | 18.6 | 3.5 | 8.4 | 徐放性コーティング |

### 5.3 スクリーニング手順

1. APIのHSP値を文献またはグループ寄与法で取得
2. 候補賦形剤のHSP値をデータベースから取得
3. 各API-賦形剤ペアのRa/REDを計算
4. RED < 1.0 の組合せを候補としてリストアップ
5. 候補について実験的検証（DSC、XRD等）を実施

## 6. 実装における設計判断

### 6.1 薬物エンティティの取扱い

薬物はSolventエンティティとして扱う（液体溶媒と同じHSPデータ構造を使用）。これは以下の理由による：

- 薬物分子のHSP値はバルク凝集エネルギーから定義され、溶質としての性質を表す
- Hansen球法で決定されたR₀値を持つ場合がある
- 既存のRa/RED計算ロジックをそのまま適用可能

### 6.2 溶解性分類閾値

薬物溶解性の分類はナノ粒子分散性と同じ方向性（RED小=良好）であり、同様の閾値を適用：

```
HIGH_SOLUBILITY:    RED < 0.5
SOLUBLE:            0.5 ≤ RED < 1.0
LIMITED:            1.0 ≤ RED < 1.5
INSOLUBLE:          RED ≥ 1.5
```

### 6.3 水系特殊処理

水系の予測精度の問題から、水が溶媒として選択された場合に注意喚起メッセージを表示する設計を検討：

```
"水系のHSP予測は精度が限定的です。
 水の特殊な水素結合特性により、実際の溶解度は予測値と
 大きく異なる場合があります。"
```

## 7. 限界事項と注意点

### 7.1 結晶充填効果（多形）

HSPは分子間相互作用のエネルギーを反映するが、結晶格子エネルギーは考慮しない。

```
溶解度 = f(分子間相互作用) + f(結晶格子エネルギー) + f(エントロピー)
```

HSPは第1項のみを評価する。多形（polymorphism）により同じ化合物でも結晶形が異なれば溶解度が異なるが、HSPでは区別できない。

**例:** リトナビルの結晶形I（不安定型）と結晶形II（安定型）では溶解度が数倍異なるが、HSP値は同一。

### 7.2 イオン化・pH効果

多くの薬物は弱酸性または弱塩基性であり、pHによりイオン化状態が変化する：

```
Henderson-Hasselbalch式:
弱酸: pH = pKa + log([A⁻]/[HA])
弱塩基: pH = pKa + log([B]/[BH⁺])
```

イオン形の溶解度は非イオン形の10-1000倍であることが多いが、HSPは非イオン形のみを評価する。

### 7.3 非晶質 vs 結晶質

非晶質（アモルファス）状態の薬物は結晶質よりも高い溶解度を示す（一般に1.5-10倍）。固体分散体やホットメルト押出法で調製される非晶質薬物の溶解度はHSPのみでは予測困難。

### 7.4 溶媒和物・水和物

一部の薬物は溶媒和物や水和物を形成し、これが溶解度に影響する。テオフィリンの無水物と一水和物では溶解度が異なるが、HSPでは区別できない。

### 7.5 定量的溶解度の非予測

HSP/REDは溶解性の定性的（溶ける/溶けない）または半定量的（相対的な溶解性の序列）予測に留まり、mg/mL単位の定量的溶解度を直接予測することはできない。

## 9. 実装判断のまとめ

| 判断項目 | 決定 | 理由 |
|---------|------|------|
| 予測指標 | RED（Ra/R₀） | 既存のHSP計算基盤を活用 |
| 溶解性分類 | 4段階 | 医薬品開発の判断粒度に合致 |
| 薬物エンティティ | Solventとして扱う | 既存データ構造を再利用 |
| 水系の注意喚起 | 実装 | 水系の予測精度限界を明示 |
| 定量的予測 | 非対応 | HSPの理論的限界 |
| 結晶多形・イオン化 | 非考慮（限界として明記） | HSPモデルの範囲外 |
| 主要用途 | 有機溶媒選定のスクリーニング | 水系より有機溶媒系で高精度 |
