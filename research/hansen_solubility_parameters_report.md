# Hansen溶解度パラメータ（HSP）調査レポート

調査日: 2026-03-15

---

## 1. エグゼクティブサマリー

Hansen溶解度パラメータ（HSP）は、物質間の溶解性・相溶性を予測するための定量的フレームワークである。1967年にCharles M. Hansenが博士論文で提唱し、Hildebrandの溶解度パラメータ（SP値）を**分散力（δD）**、**極性（δP）**、**水素結合（δH）**の3成分に分解することで、より精密な溶解性予測を可能にした。現在、塗料・コーティング、医薬品、ナノ材料、化粧品、食品科学など幅広い分野で活用されている。

---

## 2. 定義と理論的基盤

### 2.1 Hildebrand溶解度パラメータ（SP値）との関係

Hildebrandの溶解度パラメータ（δ）は、物質の凝集エネルギー密度の平方根として定義される：

```
δ = √(ΔHv - RT) / Vm
```

- ΔHv: 蒸発エンタルピー
- R: 気体定数
- T: 絶対温度
- Vm: モル体積

SP値は単一の数値であり、非極性物質間の溶解性予測には有効だが、極性物質や水素結合を持つ物質の溶解性予測には限界がある。

### 2.2 Hansen溶解度パラメータの3成分

Hansenは、全凝集エネルギー（E）を3つの分子間相互作用に分解した：

```
E = Ed + Ep + Eh
```

これをモル体積（V）で割り、平方根を取ることで3つのパラメータが得られる：

| パラメータ | 記号 | 由来する分子間力 | 説明 |
|-----------|------|----------------|------|
| 分散力項 | δD | ロンドン分散力 | 分子間の一時的な双極子による引力 |
| 極性項 | δP | 双極子-双極子相互作用 | 永久双極子間の引力 |
| 水素結合項 | δH | 水素結合 | 水素結合ドナー/アクセプター間の引力 |

全溶解度パラメータとの関係：

```
δ² = δD² + δP² + δH²
```

単位はすべて **MPa^(1/2)**（または同等の (cal/cm³)^(1/2); 1 (cal/cm³)^(1/2) ≈ 2.0455 MPa^(1/2)）

### 2.3 Hansen空間とHansen球

3つのパラメータ（δD, δP, δH）を三次元空間の座標として表現したものを**Hansen空間**と呼ぶ。

#### Hansen距離（Ra）

2つの物質間の相溶性は、Hansen空間内の距離（Ra）で評価される：

```
Ra² = 4(δD1 - δD2)² + (δP1 - δP2)² + (δH1 - δH2)²
```

**注意**: δDの係数が4であるのは、実験データとの整合性を高めるための経験的補正である。

#### 相互作用半径（R₀）とRED

溶質（ポリマー等）には固有の**相互作用半径（R₀）**があり、Hansen空間上でR₀を半径とする球（**Hansen球**）を描く。この球の内側に位置する溶媒は良溶媒、外側は貧溶媒と判定される。

**相対エネルギー差（RED）**：
```
RED = Ra / R₀
```

| RED値 | 解釈 |
|-------|------|
| RED < 1 | 溶解する（良溶媒） |
| RED ≈ 1 | 境界領域（部分溶解） |
| RED > 1 | 溶解しない（貧溶媒） |

## 4. 測定・計算方法

### 4.1 実験的手法：Hansen球法

1. 対象試料を、HSP値が既知の20〜30種類の溶媒に混合する
2. 各溶媒における溶解/不溶解を判定する
3. 結果をHansen空間（δD, δP, δH）にプロットする
4. 溶解した溶媒を含み、溶解しなかった溶媒を含まない最適球（Hansen球）を算出する
5. 球の中心がHSP値、半径がR₀となる

### 4.2 計算的手法

| 手法 | 概要 | 精度 |
|------|------|------|
| **基貢献法（Group Contribution）** | 分子を官能基に分解し、各基の寄与を合算 | 中程度。新規構造には限界あり |
| **HSPiPソフトウェア** | Stefanis-Panayiotou法等を実装。1200+化学物質のデータベースを搭載 | 高い。商用標準ツール |
| **分子動力学シミュレーション** | MD計算で凝集エネルギー密度を直接算出 | 高いが計算コストも大きい |
| **逆ガスクロマトグラフィー（IGC）** | 固体試料のHSP測定に有効 | 実験条件に依存 |

### 4.3 混合溶媒のHSP計算

混合溶媒のHSP値は、各成分の**体積分率**による加重平均で求められる：

```
δD_mix = Σ(φi × δDi)
δP_mix = Σ(φi × δPi)
δH_mix = Σ(φi × δHi)
```

φi: 成分iの体積分率

## 5. 応用事例

### 5.1 塗料・コーティング（HSPの原点）

**課題**: 適切な溶媒の選択は塗膜品質に直結する

**HSPの活用**:
- 樹脂（バインダー）のHSP値と溶媒のHSP値を比較し、最適な溶媒系を設計
- 環境規制による有害溶媒の代替（例: VOC規制対応で混合溶媒を設計）
- 塗膜の耐溶剤性予測

**具体例**: エポキシ樹脂コーティングの溶媒選択
- エポキシ樹脂のHSP: δD=20.0, δP=12.0, δH=11.5
- キシレン単独では不十分（RED > 1）だが、キシレン + ブタノールの混合溶媒で最適化可能

### 5.2 医薬品製剤

**課題**: 難溶性薬物の溶解度向上と製剤設計

**HSPの活用**:
- 薬物-賦形剤の相溶性予測（固体分散体の設計）
- 経皮吸収（DDS）：薬物と皮膚角質層のHSP適合性評価
- コーティング膜処方の最適化

**具体例**: 固体分散体の賦形剤選択
- 難溶性薬物のHSP値と各種高分子賦形剤（PVP, HPMC, PEG等）のHSP値を比較
- RED < 1の組み合わせを選択することで、安定な非晶質固体分散体を設計

### 5.3 ナノ材料の分散

**課題**: カーボンナノチューブ（CNT）やグラフェンの均一分散

**HSPの活用**:
- ナノ材料のHSP値を実験的に決定し、最適分散溶媒を選択
- 表面修飾によるHSP値の調整

**具体例**: カーボンナノチューブの分散
- SWCNTのHSP: δD≈17.8, δP≈7.5, δH≈7.6（報告値は文献により変動）
- NMP（N-メチルピロリドン）やDMFが良好な分散媒体（HSP値が近い）

### 5.4 プラスチックの耐薬品性評価

**課題**: プラスチック製品が接触する化学物質に対する耐性の予測

**HSPの活用**:
- ゴム手袋の化学物質透過性評価
- 食品包装材のバリア性能予測
- 環境応力亀裂（ESC）の予測

**具体例**: フッ素ゴム（FKM）の耐溶剤性
- FKMのHSP値と各種溶剤のHSP値の距離（Ra）を計算
- Ra > R₀（RED > 1）の溶剤に対しては良好な耐性を示す

### 5.5 化粧品開発

**課題**: 有効成分の溶解・浸透性最適化

**HSPの活用**:
- 皮膚角質層のHSP値（δD≈17.2, δP≈8.8, δH≈12.5）との適合性評価
- エマルション系の安定性予測

### 5.6 接着剤設計

**課題**: 接着対象物との親和性最適化

**HSPの活用**:
- 被着体表面のHSP値と接着剤のHSP値の距離を最小化
- プライマー選択の指針

## 6. リスクと注意事項

| 限界・注意点 | 説明 |
|-------------|------|
| **温度依存性** | HSP値は温度によって変化する。標準的なデータは25°Cで取得されており、高温・低温条件での適用には補正が必要 |
| **結晶性の影響** | HSPは非晶質状態の相互作用を前提としており、高い結晶性を持つ物質では予測精度が低下する |
| **分子量の影響** | ポリマーの溶解性は分子量にも依存するが、HSP自体は分子量を考慮しない |
| **架橋構造** | 架橋ポリマーは膨潤するが溶解しない場合があり、溶解/不溶解の二値判定が困難 |
| **データの信頼性** | 文献値は測定法や条件により変動がある。特にポリマーのHSP値は報告者により差がある |
| **動力学的因子** | HSPは熱力学的パラメータであり、溶解速度や拡散速度は別途考慮が必要 |

## 8. 推奨事項

1. **溶媒選択・材料設計への導入**: HSPは溶媒選択の試行錯誤を大幅に削減できる。特に塗料・コーティング、医薬品製剤開発においてROIが高い
2. **HSPiPの活用検討**: 商用ソフトウェアだが、広範なデータベースと計算ツールにより、効率的なHSP活用が可能
3. **実験との組み合わせ**: HSPは予測ツールであり、最終的には実験的検証が不可欠。予備スクリーニングとしてHSPを活用し、候補を絞った上で実験に進むのが効率的
4. **混合溶媒設計**: 単一溶媒で最適なHSP適合が得られない場合、混合溶媒の活用が有効。環境規制対応にも有用
