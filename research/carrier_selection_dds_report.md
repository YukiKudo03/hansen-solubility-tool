# HSPによる薬物送達キャリア選定（DDS） 調査レポート

調査日: 2026-03-15

---

## 1. エグゼクティブサマリー

Hansen溶解度パラメータ（HSP）を用いた薬物送達システム（DDS: Drug Delivery System）におけるキャリア材料の選定手法を調査した。薬物とキャリア間のHSP親和性は、カプセル化効率（encapsulation efficiency）および薬物充填量（drug loading）の重要な予測因子である。RED（相対エネルギー差）が小さいほど薬物-キャリア間の親和性が高く、高い封入効率が期待できる。

**重要な設計判断:** RED計算において `carrier.r0`（キャリアのHansen球半径）を使用する。キャリアがHansen球のオーナーであり、薬物がそのHansen球に入るかどうかを評価する構図となる。これはキャリアがPartエンティティ、薬物がSolventエンティティに対応する。

**結論:** HSPベースのキャリア選定は、実験的スクリーニング（DSC、溶解試験等）の前段階として有効であり、候補キャリアの絞り込みに大幅な時間短縮をもたらす。

## 2. 理論的背景

### 2.1 DDSの概要

薬物送達システムは、薬物を体内の標的部位に効率的に送達するための技術体系である。キャリア材料は薬物を封入・保護し、制御された放出を実現する。

**主なキャリア形態:**

| キャリア形態 | サイズ | 材料例 | 特徴 |
|-------------|--------|--------|------|
| ナノ粒子 | 10-1000 nm | PLGA, PLA, キトサン | EPR効果による腫瘍集積 |
| マイクロ粒子 | 1-1000 μm | PLGA, ゼラチン | 徐放性製剤 |
| リポソーム | 50-500 nm | リン脂質 | 細胞膜類似構造 |
| ポリマーマトリクス | マクロ | HPMC, エチルセルロース | 経口徐放錠 |
| ミセル | 10-100 nm | PEG-PLA, PEG-PCL | 両親媒性ブロックコポリマー |
| デンドリマー | 1-10 nm | PAMAM | 精密な分子設計 |

### 2.2 薬物-キャリア親和性の重要性

薬物-キャリア間の親和性はDDS設計において以下を決定する：

1. **カプセル化効率（EE%）**: 投入薬物量に対して実際にキャリアに封入された量の割合
2. **薬物充填量（DL%）**: キャリア全体に対する薬物の質量割合
3. **放出プロファイル**: 初期バースト放出の有無と放出速度
4. **物理的安定性**: 保存中の薬物析出・結晶化の抑制

```
高い親和性（RED小）→ 高EE% + 高DL% + 緩やかな放出 + 安定
低い親和性（RED大）→ 低EE% + 低DL% + 急速放出 + 不安定
```

### 2.3 HSPによる薬物-キャリア親和性評価

```
Ra² = 4·(δD_drug − δD_carrier)² + (δP_drug − δP_carrier)² + (δH_drug − δH_carrier)²
RED = Ra / R₀_carrier
```

**CRITICAL: `R₀_carrier`（キャリアのHansen球半径）を使用**

キャリアはポリマーまたは脂質であり、Hansen球のオーナーである。薬物（低分子）がキャリアのHansen球内に位置するかどうかを評価する。これは以下の理由による：

1. **物理的意味**: キャリアポリマーが「溶解する」物質の範囲がHansen球で定義される。薬物がこの球内にあれば、キャリアに混和する
2. **データの可用性**: ポリマーのR₀値は文献で確立されている。低分子薬物のR₀値は不定義の場合が多い
3. **実装の一貫性**: Part（キャリア）vs Solvent（薬物）の構図でRa/RED計算を統一的に処理

### 2.4 Flory-Huggins χパラメータとの関連

薬物-ポリマー混和性はFlory-Huggins χパラメータでも評価できる：

```
χ_drug-carrier = V_drug · (δ_drug − δ_carrier)² / (RT)
```

3次元HSP版：

```
χ_drug-carrier ≈ V_drug · [4·(ΔδD)² + (ΔδP)² + (ΔδH)²] / (4·RT)
```

| χ値 | 混和性 | DDS設計への意味 |
|-----|--------|---------------|
| χ < 0.5 | 混和 | 安定な薬物-キャリア複合体 |
| 0.5 ≤ χ < 1.0 | 限定的 | 低薬物充填量なら安定 |
| χ > 1.0 | 非混和 | 相分離・薬物析出リスク |

REDとχは相関するが、REDの方がV_drug（薬物のモル体積）が不要であり、実用的である。

## 4. キャリアをPartエンティティとして扱う根拠

### 4.1 設計の一貫性

本ツールのエンティティ設計：

| エンティティ | 役割 | DDS文脈での対応 |
|-------------|------|---------------|
| Part | Hansen球のオーナー（R₀を持つ） | **キャリア材料** |
| Solvent | 評価対象（Hansen球に入るか判定） | **薬物** |

キャリアはポリマーまたは脂質であり、以下の点でPartの定義に合致する：

1. **HSP値とR₀値の両方を持つ**: 文献で確立されたデータが利用可能
2. **Hansen球で材料特性が定義される**: 球の内部=相溶する物質群
3. **受動的な評価フレーム**: 「薬物がキャリアのHansen球に入るか」を評価

### 4.2 薬物のSolventとしての取扱い

薬物（API）は低分子化合物であり、以下の点でSolventの定義に合致する：

1. **HSP値を持つ**（δD, δP, δH）
2. **R₀値は不定義のことが多い**（Hansen球法で決定されていない場合）
3. **キャリアに対する「溶質」**として振る舞う

## 5. 実用的計算例

### 例1: イブプロフェンの最適キャリア選定

イブプロフェン: δD=17.6, δP=3.7, δH=7.0

| キャリア | Ra | R₀ | RED | 親和性評価 |
|---------|------|-----|------|----------|
| PCL | 1.6 | 8.5 | 0.19 | 優秀 |
| PLA | 4.3 | 9.7 | 0.44 | 優秀 |
| エチルセルロース | 3.1 | 7.5 | 0.41 | 優秀 |
| PLGA | 6.8 | 10.5 | 0.65 | 良好 |
| Eudragit RS | 3.6 | 7.5 | 0.48 | 優秀 |
| HPMC | 12.1 | 10.0 | 1.21 | 不良 |
| キトサン | 17.5 | 15.0 | 1.17 | 不良 |

**解釈:**
- PCLがイブプロフェンに最も高い親和性（RED=0.19）→ 高い封入効率が期待
- PLA、エチルセルロースも優秀な親和性
- HPMC、キトサンは親和性が低い → δHの大きな差に起因
- これはイブプロフェンが疎水性（BCS Class II）であることと整合

### 例2: アセトアミノフェンの最適キャリア選定

アセトアミノフェン: δD=18.1, δP=10.2, δH=16.2

| キャリア | Ra | R₀ | RED | 親和性評価 |
|---------|------|-----|------|----------|
| PLGA | 5.0 | 10.5 | 0.48 | 優秀 |
| PLA | 10.3 | 9.7 | 1.06 | 不良 |
| HPMC | 1.4 | 10.0 | 0.14 | 優秀 |
| キトサン | 8.4 | 15.0 | 0.56 | 良好 |
| ゼラチン | 4.4 | 14.0 | 0.31 | 優秀 |
| PCL | 9.3 | 8.5 | 1.09 | 不良 |
| PEG 6000 | 5.1 | 11.0 | 0.46 | 優秀 |

**解釈:**
- HPMCがアセトアミノフェンに最も高い親和性（RED=0.14）
- ゼラチン、PEG、PLGAも優秀
- PCL、PLAは親和性が低い → アセトアミノフェンのδH=16.2とPCL/PLAの低δHの差に起因
- アセトアミノフェンは親水性（BCS Class I）であり、親水性キャリアとの親和性が高い結果は妥当

## 6. 実験的検証手法との対応

### 6.1 DSC（示差走査熱量測定）による混和性評価

薬物-キャリア混合物のDSC測定：

- **単一のTg**: 薬物とキャリアが分子レベルで混和（RED小と対応）
- **2つのTg**: 相分離（RED大と対応）
- **薬物の融解ピーク消失**: 薬物が非晶質化してキャリアに分散

### 6.2 ホットステージ顕微鏡

加熱下での薬物-キャリア混合物の直接観察：
- 薬物結晶の消失温度が低いほど親和性が高い
- 相分離の進行をリアルタイムで観察可能

### 6.3 溶解試験

薬物-キャリア複合体の溶解試験：
- 非晶質固体分散体の溶解速度向上を確認
- HSP親和性が高いほど安定な非晶質状態が維持され、溶解速度向上が持続

### 6.4 HSP予測と実験結果の対応

| HSP RED | DSC結果（予想） | 溶解試験（予想） |
|---------|---------------|----------------|
| RED < 0.5 | 単一Tg、融解ピーク消失 | 大幅な溶解速度向上 |
| 0.5-1.0 | 単一Tg（低DL%時） | 中程度の向上 |
| 1.0-1.5 | 2つのTgの可能性 | 限定的な向上 |
| RED > 1.5 | 明確な相分離 | ほとんど向上なし |

## 7. 実装における設計判断

### 7.1 R₀の使用

```
RED = Ra / carrier.r0   // ← carrier（Part）のR₀を使用
```

**NOT:**
```
RED = Ra / drug.r0       // ← 薬物のR₀は使用しない
```

理由は2.3節で述べた通り、キャリアがHansen球のオーナーであり、薬物がそのHansen球に入るかを判定するため。

### 7.2 親和性分類閾値

ナノ粒子分散性・可塑剤選定と同じ閾値体系：

```
EXCELLENT:  RED < 0.5    高い封入効率（EE > 80%予想）
GOOD:       0.5-0.8      良好な封入効率（EE 50-80%予想）
POSSIBLE:   0.8-1.0      限定的な封入効率（EE 20-50%予想）
POOR:       1.0-1.5      低い封入効率（EE < 20%予想）
INCOMPATIBLE: RED ≥ 1.5  封入困難
```

**EE%の目安は概算**であり、調製法（溶媒蒸発法、噴霧乾燥法、エマルション法等）や条件に大きく依存する。

### 7.3 結果の表示

推奨キャリアリストに加え、以下の情報を付加：

- RED値と親和性ランク
- キャリアの材料特性（生分解性、FDA承認状況）
- 推奨されるDDS形態（ナノ粒子、マイクロ粒子、マトリクス等）
- HSP各成分の差（δD, δP, δHのどの差が大きいか）→ 構造的洞察

## 8. 限界事項と注意点

### 8.1 表面修飾の影響

ナノ粒子キャリアはPEG化等の表面修飾が一般的である。表面修飾はキャリア表面のHSP値を変化させるが、バルクポリマーのHSP値には反映されない。

```
修飾前: PLGA → δD=17.4, δP=10.7, δH=12.0
修飾後: PEG-PLGA → 表面はPEG的な性質を持つがHSP値は不明確
```

### 8.2 薬物の結晶性

結晶性薬物をキャリアに封入する場合、結晶格子エネルギーを克服する必要がある。HSPは分子間相互作用のみを評価し、結晶格子エネルギーは含まない。

```
実効的な混和性 = HSP親和性 − 結晶格子エネルギーの寄与
```

高融点（= 高結晶格子エネルギー）の薬物は、REDが小さくても封入効率が予想より低くなることがある。

### 8.3 放出速度の非予測

HSP/REDは薬物-キャリア間の熱力学的親和性のみを評価し、放出速度は予測できない。放出速度は以下に依存する：

- キャリアの分解速度（PLGA: 週〜月、PLA: 月〜年）
- 粒子サイズ（小さいほど放出が速い）
- 薬物の拡散係数
- 放出環境のpH・温度・酵素

### 8.4 多成分系

実際のDDS製剤は薬物+キャリア+界面活性剤+凍結乾燥保護剤など複数成分から成る。HSPは二成分（薬物-キャリア）の親和性のみを評価し、多成分の相互作用は考慮されない。

### 8.5 in vivo環境

生体内では以下の因子が加わり、in vitro（HSP予測）との乖離が生じうる：

- 血漿タンパク質の吸着（プロテインコロナ形成）
- マクロファージによる取り込み
- pH変化（胃: pH 1-2、血液: pH 7.4）
- 酵素による分解

## 10. 実装判断のまとめ

| 判断項目 | 決定 | 理由 |
|---------|------|------|
| キャリアエンティティ | Part | HSP球のオーナーとして統一 |
| 薬物エンティティ | Solvent | 低分子、評価対象 |
| R₀の使用 | carrier.r0 | キャリアがHansen球のオーナー |
| 親和性分類 | 5段階 | 分散性・可塑剤と一貫した体系 |
| 閾値 | excellentMax=0.5, goodMax=0.8, possibleMax=1.0, poorMax=1.5 | Hansen球の物理的意味に基づく |
| EE%推算 | 概算のみ（参考値） | 調製法に強く依存 |
| 表面修飾 | 非考慮（限界として明記） | バルクHSPと表面HSPの乖離 |
| 放出速度 | 非対応（限界として明記） | 速度論はHSPの範囲外 |
