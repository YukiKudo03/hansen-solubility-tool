# HSPによる塗膜耐薬品性予測 調査レポート

調査日: 2026-03-15

---

## 1. エグゼクティブサマリー

Hansen溶解度パラメータ（HSP）を用いた塗膜（コーティング）の耐薬品性予測手法を調査した。耐薬品性の評価はポリマー溶解性評価と同じRa/RED計算を使用するが、**解釈が逆転**する点が最大の特徴である。溶解性評価ではRED < 1が「溶解する（溶ける）」を意味するのに対し、耐薬品性評価ではRED < 1は「溶媒が塗膜を侵す（耐性がない）」ことを意味する。すなわち、REDが大きいほど良好な耐薬品性を示す。

**結論:** REDベースの5段階分類により、塗膜の化学薬品に対する耐性を迅速にスクリーニングできる。moderateMaxを1.0ではなく1.2に設定した理由は、塗膜の架橋構造がHansen球の境界を超えた溶媒に対しても追加の耐性を提供するためである。

## 2. 理論的背景

### 2.1 塗膜の耐薬品性とは

塗膜（コーティング）の耐薬品性は、化学薬品の接触による塗膜の劣化・変質に対する抵抗力を指す。劣化のメカニズムは以下の段階で進行する：

```
1. 薬品分子が塗膜表面に接触
2. 薬品分子が塗膜内部に浸透・拡散
3. ポリマー鎖間に薬品分子が侵入 → 膨潤
4. さらなる浸透 → ブリスタリング（ふくれ）、剥離
5. 最終的にポリマー鎖の溶解 → 塗膜崩壊
```

### 2.2 HSP距離と耐薬品性の関係（逆向き解釈）

ポリマー溶解性と耐薬品性の関係：

```
Ra² = 4·(δD_coating − δD_chem)² + (δP_coating − δP_chem)² + (δH_coating − δH_chem)²
RED = Ra / R₀
```

**解釈の逆転:**

| RED値 | ポリマー溶解性評価 | 塗膜耐薬品性評価 |
|-------|-------------------|-----------------|
| RED < 0.5 | 確実に溶解 | **耐性なし**（確実に侵される） |
| 0.5-1.0 | 溶解する | **耐性が低い** |
| 1.0-1.5 | 限定的溶解 | **中程度の耐性** |
| RED > 1.5 | 不溶 | **良好な耐性** |
| RED > 2.0 | 完全に不溶 | **優れた耐性** |

**同じ数式・同じ計算で、望ましい方向が反転する。**

### 2.3 塗膜の破壊メカニズム

#### 膨潤（Swelling）

溶媒が塗膜に浸透し、ポリマー鎖間に入り込むことで体積が増大する。可逆的な場合もあるが、繰り返しにより永久変形が生じる。

#### ブリスタリング（Blistering）

塗膜と基材の界面に溶媒または水が蓄積し、塗膜が基材から浮き上がる現象。浸透圧差が駆動力となる。

#### 剥離（Delamination）

膨潤・ブリスタリングが進行し、塗膜が基材から完全に剥がれる。基材との密着力を超える内部応力が原因。

#### 溶解（Dissolution）

溶媒が十分に強い場合（RED << 1）、塗膜のポリマーが完全に溶解する。架橋塗膜では起こりにくいが、非架橋の熱可塑性塗膜では発生しうる。

## 3. 塗膜をPartエンティティとして扱う根拠

### 3.1 なぜPartなのか

塗膜はポリマーベースの材料であり、以下の理由からPartエンティティとして扱う：

1. **HSP値が文献で確立されている**: エポキシ、ポリウレタン、アクリル等の塗膜樹脂にはHSP値が利用可能
2. **Hansen球が定義できる**: R₀値が溶解性試験から決定されている
3. **受動的な評価対象**: 薬品（Solvent）が塗膜（Part）を「攻撃する」という構図はPart-Solventの関係に一致
4. **既存の計算ロジックを再利用**: Ra/RED計算はPart vs Solventの枠組みで統一的に処理できる

### 3.2 代表的な塗膜樹脂のHSP値

| 塗膜樹脂 | δD | δP | δH | R₀ | 主な用途 |
|---------|----|----|-----|-----|---------|
| エポキシ（ビスフェノールA型） | 20.0 | 12.0 | 11.5 | 12.7 | タンクライニング、重防食 |
| ポリウレタン（2液型） | 18.0 | 8.5 | 9.0 | 10.5 | 床材、自動車 |
| アクリル（メタクリル系） | 18.6 | 10.5 | 7.5 | 8.6 | 建築、自動車 |

| フッ素樹脂（PVDF） | 17.0 | 12.1 | 10.2 | 7.0 | 超耐候性 |
| シリコーン | 15.9 | 0.1 | 4.7 | 12.3 | 耐熱、撥水 |
| 塩化ビニル共重合体 | 18.2 | 7.5 | 3.4 | 6.2 | 防食 |

## 4. 耐薬品性分類の閾値設計

### 4.1 5段階分類

| レベル | 名称 | RED範囲 | 物理的意味 |
|--------|------|---------|-----------|
| No Resistance（耐性なし） | 塗膜崩壊 | RED < 0.5 | Hansen球の深部。溶媒が塗膜を確実に破壊 |
| Poor（不良） | 明確な劣化 | 0.5 ≤ RED < 0.8 | Hansen球内。膨潤・ブリスタリングが発生 |
| Moderate（中程度） | 限定的劣化 | 0.8 ≤ RED < 1.2 | Hansen球境界付近。軽微な影響の可能性 |
| Good（良好） | 実用的な耐性 | 1.2 ≤ RED < 2.0 | Hansen球外。実用環境で問題なし |
| Excellent（優秀） | 完全耐性 | RED ≥ 2.0 | HSP値が大きく乖離。影響なし |

### 4.2 moderateMaxが1.2である理由（1.0ではない）

通常のHSP評価ではRED = 1.0がHansen球の境界であり、これを超えれば「溶解しない」と判定する。しかし塗膜耐薬品性では1.0ではなく**1.2**をmoderate-goodの境界とした。その根拠：

1. **架橋構造の追加耐性**: 塗膜は通常、架橋（硬化）されている。架橋構造はポリマー鎖の溶媒による膨潤を物理的に制限し、Hansen球の境界（RED=1.0）付近でも追加の耐性を提供する。

2. **浸透の速度論**: RED ≈ 1.0付近の溶媒は熱力学的にはHansen球外にあるが、短時間の接触では僅かな浸透が起こりうる。実用的な「耐性あり」の判定には安全マージンが必要。

3. **実験データとの整合**: ISO 2812やASTM D1308による浸漬試験では、RED = 1.0-1.2の領域で「変化あり」と評価されるケースが報告されている。

4. **保守的設計**: 塗膜耐薬品性の評価では、リスクの過小評価（耐性があると誤判定）を避けるため、安全側に閾値を設定する。

```
                  Hansen球
                ┌─────────┐
  耐性なし ─── │ 球内部  │ ─── 不良 ─── 中程度 ─── 良好 ─── 優秀
  RED < 0.5    │         │  0.5-0.8   0.8-1.2   1.2-2.0   RED ≥ 2.0
                └─────────┘
                 RED = 1.0
                  ↑
              Hansen球境界
              （架橋の追加耐性により
               ここでは完全に安全とは限らない）
```

## 5. 試験規格との比較

### 5.1 ISO 2812（浸漬法による耐液性試験）

ISO 2812は塗膜の耐液性を評価する国際規格：

- **Part 1**: 一般的な液体への浸漬
- **Part 2**: 水浸漬
- **Part 3**: 薬品点滴法
- **Part 4**: 点滴法（スポット試験）

**評価項目:**
- 外観変化（白化、ふくれ、軟化、変色）
- 付着性の低下
- 光沢の変化

### 5.2 ASTM D1308（化学薬品による塗膜への影響）

ASTM D1308は塗膜への化学薬品の影響を評価する米国規格：

- **Method A**: オープンスポット法（薬品を塗膜上に滴下し開放）
- **Method B**: クローズドスポット法（薬品を塗膜上に滴下しカバー）

### 5.3 HSP予測との対応関係

| 試験結果 | ISO 2812等級 | HSP RED範囲（概算） |
|---------|-------------|-------------------|
| 変化なし | 0（最良） | RED > 2.0 |
| ごく僅かな変化 | 1 | 1.5-2.0 |
| 軽微な変化 | 2 | 1.2-1.5 |
| 中程度の変化 | 3 | 0.8-1.2 |
| 著しい変化 | 4 | 0.5-0.8 |
| 塗膜破壊 | 5（最悪） | RED < 0.5 |

**注意:** この対応は概算であり、実際の耐薬品性は架橋密度、膜厚、暴露時間・温度に大きく依存する。

## 6. 実用的計算例

### 例1: エポキシ塗膜の各種薬品に対する耐性評価

エポキシ塗膜: δD=20.0, δP=12.0, δH=11.5, R₀=12.7

| 薬品 | δD | δP | δH | Ra | RED | 耐性評価 |
|------|----|----|-----|------|------|----------|
| 水 | 15.5 | 16.0 | 42.3 | 32.1 | 2.53 | Excellent |
| エタノール | 15.8 | 8.8 | 19.4 | 12.2 | 0.96 | Moderate |
| アセトン | 15.5 | 10.4 | 7.0 | 10.3 | 0.81 | Moderate |
| MEK | 16.0 | 9.0 | 5.1 | 10.8 | 0.85 | Moderate |
| トルエン | 18.0 | 1.4 | 2.0 | 13.5 | 1.06 | Moderate |
| ヘキサン | 14.9 | 0.0 | 0.0 | 16.2 | 1.28 | Good |
| DMF | 17.4 | 13.7 | 11.3 | 5.4 | 0.43 | No Resistance |
| ジクロロメタン | 17.0 | 7.3 | 7.1 | 8.0 | 0.63 | Poor |
| 酢酸エチル | 15.8 | 5.3 | 7.2 | 10.8 | 0.85 | Moderate |
| NMP | 18.0 | 12.3 | 7.2 | 5.7 | 0.45 | No Resistance |

**解釈:**
- エポキシ塗膜は水に対して優秀な耐性を示す（水タンクライニングとして実績がある）
- DMF、NMPなどの強極性溶媒には耐性がない
- ケトン類（アセトン、MEK）に対しては中程度の耐性（短時間暴露では問題ないが長時間は避ける）
- ヘキサンなどの非極性溶媒に対しては良好な耐性

### 例2: フッ素樹脂（PVDF）塗膜

PVDF: δD=17.0, δP=12.1, δH=10.2, R₀=7.0

| 薬品 | δD | δP | δH | Ra | RED | 耐性評価 |
|------|----|----|-----|------|------|----------|
| 水 | 15.5 | 16.0 | 42.3 | 32.5 | 4.64 | Excellent |
| エタノール | 15.8 | 8.8 | 19.4 | 10.1 | 1.44 | Good |
| アセトン | 15.5 | 10.4 | 7.0 | 4.6 | 0.66 | Poor |
| トルエン | 18.0 | 1.4 | 2.0 | 13.0 | 1.86 | Good |
| DMF | 17.4 | 13.7 | 11.3 | 2.1 | 0.30 | No Resistance |

PVDFは多くの薬品に優れた耐性を持つが、DMFなどの特殊溶媒には弱い。これは実務的な知見と一致する。

## 7. 応用シナリオ

### 7.1 タンクライニング選定

化学プラントにおける貯蔵タンクの内面ライニング材の選定：

1. 貯蔵する薬品のHSP値を特定
2. 候補ライニング材（エポキシ、ビニルエステル、ポリウレタン等）のHSP値を取得
3. 各組合せのREDを計算
4. RED > 1.5 の組合せを推奨候補として提示

### 7.2 床材コーティング仕様

工場床の耐薬品性コーティング選定：

1. 使用される薬品リスト（洗浄剤、潤滑油、酸・アルカリ等）を特定
2. 各薬品に対して全候補コーティングのREDを計算
3. **全ての薬品に対してREDがGood以上**の塗膜を選定

## 8. 限界事項と注意点

### 8.1 架橋密度の非考慮

架橋密度はHSPモデルでは直接反映されない。実際には：

- 高架橋密度: Hansen球境界（RED≈1.0）付近でも耐性あり
- 低架橋密度: Hansen球外（RED>1.0）でも膨潤が起こりうる

moderateMaxを1.2とすることで部分的に補正しているが、定量的ではない。

### 8.2 温度・濃度効果

- 温度上昇により拡散速度が増大し、耐性が低下
- 薬品が溶液の場合、濃度により実効的なHSP値が変化
- 高温の酸・アルカリは化学的攻撃（加水分解等）も加わり、HSPの範囲を超える

### 8.3 複合薬品暴露

実環境では複数の薬品に同時に暴露されることが多い。各薬品を個別に評価する方法では、相乗効果（一方の薬品が塗膜を膨潤させ、もう一方の浸透を促進）を考慮できない。

### 8.4 化学的攻撃

以下の攻撃メカニズムはHSP（物理的親和性）では予測できない：

- **加水分解**: エステル結合やウレタン結合の分解（酸・アルカリ）
- **酸化**: 強酸化剤による主鎖切断
- **紫外線劣化との複合**: UV暴露で脆化した塗膜は薬品に対する耐性が低下
- **電気化学的腐食**: 金属基材の腐食による塗膜の裏面からの剥離

### 8.5 膜厚と暴露時間

HSP/REDは熱力学的親和性のみを評価し、速度論を含まない：

- 薄膜（< 50 μm）は浸透が速い
- 厚膜（> 500 μm）は浸透に時間がかかるが、長期間では同じ結果
- 短時間暴露（スプラッシュ）vs 長時間浸漬で許容REDは異なる

## 10. 実装判断のまとめ

| 判断項目 | 決定 | 理由 |
|---------|------|------|
| 計算手法 | Ra/RED（ポリマー溶解性と同一） | 既存ロジックの再利用 |
| 解釈方向 | RED大=良好（逆向き） | 塗膜は溶解されない方が良い |
| 塗膜エンティティ | Part | ポリマーベース、受動的評価対象 |
| 分類段階 | 5段階 | 実務で必要な判断粒度 |
| noResistanceMax | 0.5 | Hansen球深部 = 確実に侵される |
| poorMax | 0.8 | Hansen球内 = 明確な劣化 |
| moderateMax | 1.2 | **1.0ではなく1.2**: 架橋による追加耐性 |
| goodMax | 2.0 | Hansen球外 = 実用的耐性あり |
| 架橋密度 | 非考慮（moderateMax=1.2で部分補正） | データ不足 |
| 化学的攻撃 | 非対応（注意喚起のみ） | HSPの理論的範囲外 |
