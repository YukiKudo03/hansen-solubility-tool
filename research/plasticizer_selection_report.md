# HSPによる可塑剤選定支援 調査レポート

調査日: 2026-03-15

---

## 1. エグゼクティブサマリー

Hansen溶解度パラメータ（HSP）を用いた可塑剤（plasticizer）とポリマーの相溶性評価手法を調査した。可塑剤はポリマー鎖間に挿入されてガラス転移温度（Tg）を低下させ柔軟性を付与する低分子量物質であり、その機能を発揮するにはポリマーとの十分な相溶性が必要である。HSPの「似たもの同士が溶ける」原理に基づき、RED < 1の可塑剤はポリマーのHansen球内に位置し、良好な相溶性が期待される。

**結論:** REDベースの相溶性評価は可塑剤スクリーニングの初期段階で有効だが、可塑剤の移行性（揮発・抽出）、熱安定性、規制適合性はHSP単独では評価できず、追加の検討が必要である。

---

## 2. 理論的背景

### 2.1 可塑化メカニズム

可塑化（plasticization）は以下のメカニズムで進行する：

```
1. 可塑剤分子がポリマーマトリクス内に分散
2. 可塑剤分子がポリマー鎖間に挿入（分子間力を弱める）
3. ポリマー鎖の自由体積が増加
4. 鎖のセグメント運動が容易になる
5. ガラス転移温度（Tg）が低下
6. 柔軟性・延性が向上
```

**Tg低下の定量的関係（Fox式の拡張）:**

```
1/Tg_mix = w_p/Tg_p + w_pl/Tg_pl
```

ここで：
- `Tg_mix` = 可塑化後のTg
- `w_p, w_pl` = ポリマー・可塑剤の質量分率
- `Tg_p, Tg_pl` = ポリマー・可塑剤のTg

### 2.2 HSPによる相溶性予測

可塑剤の相溶性予測は「Like dissolves like」の原理に基づく：

```
Ra² = 4·(δD_polymer − δD_plast)² + (δP_polymer − δP_plast)² + (δH_polymer − δH_plast)²
RED = Ra / R₀
```

| RED値 | 相溶性 | 実用的意味 |
|-------|--------|-----------|
| RED < 0.5 | 優秀 | 高い相溶性、大量添加可能 |
| 0.5 ≤ RED < 0.8 | 良好 | 実用的な添加量が可能 |
| 0.8 ≤ RED < 1.0 | 可能 | 限定的な添加量、ブリードアウトに注意 |
| 1.0 ≤ RED < 1.5 | 不良 | 相溶性不足、短期間で移行・分離 |
| RED ≥ 1.5 | 不可 | 明確に非相溶 |

### 2.3 可塑剤と溶媒の違い

可塑剤は本質的には「留まる溶媒」とも言えるが、以下の点で溶媒と異なる：

| 特性 | 溶媒 | 可塑剤 |
|------|------|--------|
| 目的 | 溶解・分散後に蒸発 | ポリマー中に永続的に滞在 |
| 分子量 | 低い（< 200） | 中程度（200-600） |
| 蒸気圧 | 高い | 低い |
| 沸点 | 低い〜中程度 | 高い（> 200°C） |
| 添加量 | 多量（溶液全体の50-90%） | 少量〜中量（5-50 phr） |

## 4. PVCと可塑剤の相溶性評価例

### 4.1 PVCのHSP値

```
PVC: δD = 18.2, δP = 7.5, δH = 8.3, R₀ = 6.2
```

### 4.2 各可塑剤のRED計算

| 可塑剤 | Ra | RED | 相溶性評価 |
|--------|------|------|----------|
| DOP | 3.8 | 0.61 | 良好 |
| DBP | 1.8 | 0.29 | 優秀 |
| DINP | 4.3 | 0.69 | 良好 |
| DOA | 6.2 | 1.00 | 境界（可能） |
| TBC | 3.7 | 0.60 | 良好 |
| ATBC | 4.9 | 0.79 | 可能 |
| TCP | 5.6 | 0.90 | 可能 |
| ESBO | 5.9 | 0.95 | 可能 |

**解釈:**
- DBPがPVCとの相溶性で最も優秀（RED=0.29）だが、規制対象
- DOPは良好な相溶性で広く使用されてきた実績と一致
- DOAはRED=1.0で境界的。実際にDOAは単独ではPVCとの長期相溶性に問題があることが知られており、HSP予測と整合する
- クエン酸エステル類は良好〜可能の範囲で、フタル酸エステルの代替として有望

## 5. 実装における設計判断

### 5.1 可塑剤の識別方法

可塑剤はSolventエンティティのnotesフィールドに `[可塑剤]` タグを付与して識別する。

**設計根拠:**

1. **専用エンティティの不要性**: 可塑剤のHSPデータ構造は溶媒と同一（δD, δP, δH + メタデータ）
2. **柔軟なタグ付け**: notesフィールドの自由テキストにより、追加のカテゴリ情報を格納
3. **フィルタリングの容易さ**: `[可塑剤]` タグで検索・フィルタリングが可能
4. **データベースの拡張性**: 新規可塑剤の追加が容易（Solventとして追加しタグ付け）

```
// 例: Solventエンティティ
{
  name: "DOP (DEHP)",
  deltaD: 16.6,
  deltaP: 7.0,
  deltaH: 3.1,
  notes: "[可塑剤] フタル酸ジオクチル。PVC汎用可塑剤。REACH規制対象(SVHC)。"
}
```

### 5.2 分類閾値

分散性評価と同一の閾値体系を使用：

```
interface PlasticizerThresholds {
  excellentMax: number;   // default: 0.5
  goodMax: number;        // default: 0.8
  possibleMax: number;    // default: 1.0
  poorMax: number;        // default: 1.5
}
```

**同一閾値を使用する理由:**
- 可塑剤の相溶性評価はナノ粒子分散性と同じ方向性（RED小=良好）
- Hansen球の物理的意味は共通
- ユーザーにとって一貫した評価体系が理解しやすい

### 5.3 結果表示

推奨可塑剤リストに加え、以下の情報を付加する：

- RED値と相溶性ランク
- 規制情報（REACH/RoHS該当の有無）
- 可塑剤ファミリー分類
- 注意事項（移行性、熱安定性等）

## 6. 規制動向

### 6.1 REACH規制

EU REACH規則では以下のフタル酸エステルがSVHC（高懸念物質）に指定：

| 物質 | CAS番号 | SVHC理由 | 認可リスト |
|------|---------|---------|-----------|
| DEHP (DOP) | 117-81-7 | 生殖毒性 | Annex XIV |
| DBP | 84-74-2 | 生殖毒性 | Annex XIV |
| BBP | 85-68-7 | 生殖毒性 | Annex XIV |
| DIBP | 84-69-5 | 生殖毒性 | Annex XIV |

### 6.2 RoHS指令

2019年7月からRoHS指令（2011/65/EU）の規制物質にフタル酸エステル4種が追加：

- DEHP: 0.1 wt%以下
- DBP: 0.1 wt%以下
- BBP: 0.1 wt%以下
- DIBP: 0.1 wt%以下

### 6.3 食品接触規制

食品接触材料には各国の規制が適用される：

| 規制 | 地域 | 主な要件 |
|------|------|---------|
| FDA 21 CFR | 米国 | 食品接触に使用可能な物質リスト |
| EU 10/2011 | EU | プラスチック製食品接触材料規則 |
| 食品衛生法 | 日本 | ポジティブリスト制度（2020年〜） |

**HSPツールでの対応:** 規制情報はnotesフィールドに記載し、ユーザーの注意を喚起する。HSPによる相溶性評価と規制適合性は独立した判断軸であり、両方を満たす可塑剤を選定する必要がある。

## 7. 可塑剤移行性の問題

### 7.1 移行メカニズム

可塑剤の移行（migration）は以下の経路で発生する：

1. **揮発**: 可塑剤が気相に蒸発（温度・蒸気圧依存）
2. **抽出**: 接触する液体（油、水、溶剤）への可塑剤の溶出
3. **接触移行**: 接触する他の材料への可塑剤の移動（拡散）

### 7.2 HSPと移行性の関係

HSPの観点からは、可塑剤のREDが小さいほど相溶性が高く、**ポリマー中に留まりやすい（移行しにくい）**と予想される。しかし実際の移行性は以下の因子にも強く依存する：

| 因子 | 影響 | HSPで考慮 |
|------|------|----------|
| 分子量 | 高分子量ほど移行しにくい | 不可 |
| 蒸気圧 | 低蒸気圧ほど揮発しにくい | 不可 |
| 粘度 | 高粘度ほど拡散が遅い | 不可 |
| ポリマーとの親和性 | RED小ほど留まりやすい | **可能** |
| 温度 | 高温ほど移行が加速 | 間接的のみ |

**結論:** HSPは可塑剤の相溶性（ポリマーに留まる熱力学的傾向）のみを評価し、移行の速度論的側面は予測できない。

### 7.3 ブリードアウト（表面移行）

RED > 0.8 の可塑剤は長期的にポリマー表面に移行し、表面に析出する（ブリードアウト）リスクが高まる。これは以下の問題を引き起こす：

- 接触する他材料の汚染
- 外観の悪化（白色粉末状の析出）
- 可塑化効果の低下（硬化）

## 8. 限界事項と注意点

### 8.1 可塑化効率の非予測

HSP/REDは相溶性のみを評価し、可塑化の効率（同じ添加量でのTg低下量）は予測できない。可塑化効率は分子構造（主鎖の柔軟性、極性基の配置）にも依存する。

### 8.2 熱安定性

高温加工（例: PVC押出 180-200°C）時の可塑剤の分解・揮発はHSPでは評価できない。

### 8.3 二次可塑剤（エクステンダー）

一次可塑剤（ポリマーとの相溶性が高い）に加え、二次可塑剤（コスト低減目的で一次可塑剤と併用）の複合効果はHSP線形混合則で近似できるが、精度は限定的。

### 8.4 相溶性の温度依存性

HSP値は温度依存性を持つため、常温で相溶していても高温または低温で相分離が起こりうる。特に低温環境での可塑剤のブリードアウトは実務上重要な問題である。

### 8.5 長期安定性

HSPは熱力学的平衡状態を予測するが、数年〜数十年スケールの長期安定性（酸化劣化、UV劣化、加水分解）は別途評価が必要。

## 10. 実装判断のまとめ

| 判断項目 | 決定 | 理由 |
|---------|------|------|
| 可塑剤識別 | Solvent.notesに `[可塑剤]` タグ | 既存データ構造の再利用、柔軟性 |
| 相溶性評価 | RED（Ra/R₀） | ナノ粒子分散性と同じ方向性・閾値 |
| 分類段階 | 5段階（Excellent/Good/Possible/Poor/Incompatible） | 分散性と一貫した体系 |
| 閾値 | excellentMax=0.5, goodMax=0.8, possibleMax=1.0, poorMax=1.5 | Hansen球の物理的意味に基づく |
| 規制情報 | notesフィールドに記載 | HSPとは独立した判断軸 |
| 移行性予測 | 非対応（限界として明記） | HSPモデルの範囲外 |
| 熱安定性 | 非対応（限界として明記） | HSPモデルの範囲外 |
